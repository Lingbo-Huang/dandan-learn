# D7 · Wisdom Week 5 综合实战：概率日记

> 把概率思维从"道理"变成"习惯"，需要系统练习。

---

## 本周知识回顾

| 工具 | 核心洞见 |
|------|---------|
| 概率思维 | 用数字替代"可能/应该"的模糊判断 |
| 贝叶斯更新 | 新证据应该改变信念的程度 |
| 基率忽视 | 先看整体基率，再看具体细节 |
| 过度自信 | 大多数人对自己的判断过于确定 |
| 校准 | 持续追踪，让主观概率靠近客观频率 |
| 期望值决策 | 不追求"可能的好结果"，追求"期望值最高的方案" |


## 本周最重要的一课

**概率思维的本质不是计算，而是谦逊。**

承认不确定性，但仍然行动。

这是最难做到的平衡：
- 没有概率意识的人：过度自信，系统性做出错误决策
- 过于强调不确定性的人：瘫痪，无法行动

量化人的黄金标准：
> "我不知道市场明天怎么走，但我知道我的因子在历史上有 55% 的概率预测正确，所以我应该配 X 的仓位，并设置合理的止损。"
