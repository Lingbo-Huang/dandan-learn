## 综合案例：你的学习系统

用系统图分析你的学习系统。

**元素**：
- 学习时间投入
- 理解深度
- 项目质量
- 面试结果/职业反馈
- 信心
- 焦虑程度

**关系（画连线）**：
- 学习时间 → 理解深度（正）
- 理解深度 → 项目质量（正）
- 项目质量 → 面试结果（正）
- 面试结果 → 信心（正）
- 信心 → 学习时间（正？负？取决于性格）
- 焦虑 → 学习时间（正？负？取决于焦虑程度）

**反馈回路**：
- 学习 → 理解 → 信心 → 学习（正反馈！成功螺旋）
- 失败 → 焦虑 → 学习时间减少 → 更差结果（正反馈！失败螺旋）

**杠杆点**：
- 弱：增加每天学习时间（调参）
- 中：改变学习方法（从看书到做项目）
- 强：改变学习目标（从"应付面试"到"真正理解"）
- 最强：改变信念（从"我可能不适合量化"到"量化能力是可以后天培养的"）

**你的行动计划**：基于系统分析，你选择在哪个杠杆点干预？
