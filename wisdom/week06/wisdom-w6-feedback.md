# D2 · 正负反馈回路

> 正反馈让系统爆炸或崩溃，负反馈让系统稳定。理解两者，才能预测系统行为。

---

## 1. 正反馈（Reinforcing Loops）

**正反馈**：A 增加 → B 增加 → A 继续增加（自我强化）

金融例子：
- 股价上涨 → 吸引追涨者 → 更多买入 → 股价继续上涨（泡沫形成）
- 动量策略盈利 → 吸引更多资金 → 策略容量耗尽 → alpha 消失

正反馈的结果：
- 短期：强劲趋势
- 长期：必然有终点（物理极限或外部冲击）

---

## 2. 负反馈（Balancing Loops）

**负反馈**：A 增加 → B 增加 → 这反过来压低 A（自我调节）

金融例子：
- 价格高于均值 → 价值投资者卖出 → 价格回归均值
- 高利率 → 借贷成本高 → 经济降温 → 央行降息 → 利率下降

负反馈的结果：稳定性和均值回归

---

## 3. 量化策略中的反馈机制

**套利压力（负反馈）**：
- 因子有效 → 更多资金使用这个因子 → 价差压缩 → 因子失效

这是所有量化 alpha 衰减的根本机制。

**市场恐慌（正反馈）**：
- 价格下跌 → 风控触发止损 → 更多卖出 → 价格继续下跌 → 更多止损

2020年3月和2015年A股暴跌都有这个机制的身影。

---

## 4. 识别反馈回路的方法

对每对关键变量，问：
1. 当 A 增加时，B 增加还是减少？
2. 当 B 增加时，它对 A 的影响是什么？
3. 这形成了什么类型的回路？
