# D6 · 系统思维在量化中的应用

> 把本周的系统思维工具箱，应用到量化研究和策略开发。

---

## 1. 量化策略是一个系统

量化策略不只是一个公式，它是一个有反馈机制的系统：

```
市场行情 → 信号计算 → 组合构建 → 执行 → 收益 → 再分配资金 → 影响市场行情
```

每个环节都有反馈：
- 策略规模越大 → 冲击成本越高 → 收益下降（负反馈）
- 策略有效 → 吸引模仿 → alpha 衰减（负反馈）

---

## 2. 避免系统干预的陷阱

**陷阱 1：过拟合 = 调了错误的杠杆点**

当因子 IC 下降，不应该调整因子的具体参数，而应该回到经济学逻辑检验假设。

**陷阱 2：增加复杂度不总是有帮助**

系统越复杂，出错的地方越多，调试越困难。简单稳健 > 复杂精巧。

**陷阱 3：忽视策略对市场本身的影响**

当你的策略规模大到足以影响市场时，策略的信号本身被市场吸收了。

---

## 3. 系统视角下的量化研究生态

量化研究是一个多人参与的生态系统：
- 研究者发现 alpha → 套利者复制 → alpha 消失（负反馈）
- 论文发表加速了 alpha 消失（信息流传速度变量）

**结论**：
- 公开的 alpha 会快速消失，真正的边缘来自非公开的数据/方法
- 保护 alpha 的时间窗口很重要（发现到部署的速度）
- 持续创新是在这个生态中生存的必要条件

---
