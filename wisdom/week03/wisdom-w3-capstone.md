## 一、判断你在哪个阶段

**判断框架**：

对任何你感兴趣的领域（职业/行业/投资），问这些问题：

```
1. 主流媒体在大量报道吗？
   → 是：可能在期望峰值
   → 否：可能在早期或幻灭期

2. 你的非专业朋友开始讨论它了吗？
   → 是：泡沫警告
   → 否：可能还在早期

3. 这个领域的一线从业者还在创造真实价值吗？
   → 是：底层逻辑健康
   → 否：只是在分配财富，不在创造财富

4. 对比5年前，这个领域有没有变得官僚化？
   → 是：可能在过度期
   → 否：还在成长期

5. 领域里最优秀的人还在吗？
   → 是：健康
   → 否：才华已经去下一个领域了
```

## 二、周期律指导下的职业规划

**20多岁（建立期）**：
- 这是你的个人"建立期"
- 最重要的：选对赛道（未来10年的成长领域）
- 次要的：薪酬、地位（这些会来的）
- 危险信号：选了一个正在衰退期的行业

**30多岁（繁荣期）**：
- 快速积累：能力、人脉、资源
- 建立不可替代性
- 开始思考：这个领域的周期到哪了？

**40多岁（判断期）**：
- 关键决策：继续深耕，还是跳到下一个 S 曲线？
- 路径依赖的风险最高（收入高，切换代价大）
- 最好的人才：会在这个时候主动切换

## 三、周期律指导下的投资原则

**第一原则：不要在所有人都乐观时持有，不要在所有人都悲观时卖出**

```
周期 vs. 情绪：

繁荣期：大家都赚钱 → 市场情绪 = 乐观
过度期：还在赚钱 → 市场情绪 = 极度乐观（危险！）
衰退期：开始亏钱 → 市场情绪 = 恐慌（机会！）
低谷期：持续亏钱 → 市场情绪 = 绝望（最大机会）
```

**第二原则：区分基础设施和应用层**

历史规律：
- 泡沫期间，基础设施（算力、网络）会被过度建设
- 泡沫破裂后，基础设施留下来，应用层在上面重建
- **卖铲子的人比淘金者活得更好**

互联网时代：Amazon（云计算）比大多数互联网应用活得更长。
AI 时代：英伟达（算力）可能比大多数 AI 应用更持久。

**第三原则：保留安全边际**

即使判断对了，时机也可能错：
- 泡沫比你想象的更持久
- 崩溃比你想象的更剧烈
- **永远不要满仓在一个判断上**

## 四、一个思考工具：时间尺度分层

```
10年+（结构性）：
  问：大的历史趋势是什么？
  → 人口结构、技术范式、地缘政治格局
  → 这些变化慢，但不可逆

3-10年（周期性）：
  问：我们在哪个经济/技术周期？
  → 牛熊市、行业 S 曲线阶段
  → 判断当前是繁荣期还是过度期

1-3年（战术性）：
  问：当前的机会和风险是什么？
  → 具体公司、具体技术、具体政策
  → 最难预测，但最影响短期

原则：不要用短期视角做长期决策，
      也不要用长期视角逃避短期执行
```

---

## 五、最重要的认识：个人如何超越周期

历史上有少数人能够超越周期，他们的共同点：

1. **持续学习**：在每一个周期中都获得新能力，而不只是财富
2. **保持流动性**：不把自己完全固定在一个领域/地区
3. **建立真实价值**：做对世界真正有用的事，而不是只参与分配游戏
4. **长期思维 + 短期灵活**：知道大方向，但随时调整战术

**达里奥的个人总结**：
> "我最重要的认识是：在每一个阶段，都要理解游戏规则正在如何变化，而不是依赖之前有效的规则。"
