# Agent Week 4：复杂规划与推理

> **系列**：Agent + Claw 线  
> **周次**：Week 4  
> **前置**：Week 1-3 Agent 基础与主流框架

---

## 本周主题

这周进入 Agent 能力的核心领域：**复杂推理与规划**。掌握 CoT、ToT、ReAct 进阶、自我反思和任务分解，让你的 Agent 从"会执行"升级为"会思考"。

| 天 | 主题 | 关键技术 |
|----|------|----------|
| D1 | Chain-of-Thought 进阶 | CoT、Self-consistency、Zero-shot CoT |
| D2 | Tree of Thoughts | ToT 搜索算法、分支评估 |
| D3 | ReAct 进阶与变体 | Reflexion、ReAct+Critic |
| D4 | 自我反思与批评 | Self-Reflection、Constitutional AI |
| D5 | 任务分解与规划 | HTN、PDDL 简介、LLM Planner |
| D6 | 推理链组合实战 | 多种推理策略组合 |
| D7 | Capstone：复杂问题求解 Agent | 综合所有推理技术 |

## 本周目标

- [ ] 理解 CoT / ToT / ReAct 的区别与适用场景
- [ ] 能独立实现基于 Reflexion 的自我改进 Agent
- [ ] 掌握任务分解策略，能处理 5 步以上的复杂任务
- [ ] 实现一个能通过自我批评改善输出质量的 Agent
