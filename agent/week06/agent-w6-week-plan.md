## 本周主题

记忆是让 Agent 从"工具"变成"同事"的关键。这周深入 Agent 的各类记忆系统：短期、长期、情景、语义，以及背后的向量存储技术。

| 天 | 主题 |
|----|------|
| D1 | 记忆系统概论：四种记忆类型 |
| D2 | 短期记忆：上下文窗口优化 |
| D3 | 向量数据库：ChromaDB / Pinecone 实战 |
| D4 | 长期记忆：持久化存储与检索 |
| D5 | 情景记忆：事件序列与时序推理 |
| D6 | 语义记忆：知识图谱与结构化存储 |
| D7 | Capstone：有记忆的个人助手 Agent |

## 本周目标

- [ ] 理解四种记忆类型及适用场景
- [ ] 能独立搭建向量数据库并实现语义搜索
- [ ] 构建一个能记住用户偏好和历史对话的 Agent
- [ ] 掌握记忆压缩和遗忘策略
