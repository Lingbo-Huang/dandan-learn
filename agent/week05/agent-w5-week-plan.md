## 本周主题

单个 Agent 能力有限，真正的生产级系统需要**多个 Agent 协同**。这周深入多 Agent 架构：通信协议、角色分工、协作框架，以及 AutoGen 的深度实战。

| 天 | 主题 |
|----|------|
| D1 | 多 Agent 架构设计原则 |
| D2 | Agent 通信协议（消息总线/事件驱动） |
| D3 | 角色分工与协作模式 |
| D4 | AutoGen 深入实战 |
| D5 | CrewAI 与 LangGraph |
| D6 | 多 Agent 调试与监控 |
| D7 | Capstone：多 Agent 协作系统 |

## 本周目标

- [ ] 理解 Hierarchical/Flat/Pipeline 三种多 Agent 架构
- [ ] 能用 AutoGen 构建有效的多 Agent 对话系统
- [ ] 掌握 Agent 间通信的设计模式
- [ ] 实现一个 3+ 个 Agent 协作完成复杂任务的系统
