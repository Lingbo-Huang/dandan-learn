## 📚 核心知识点

### 1. 市场结构

**香港联合交易所（HKEX）**是港股的核心交易场所，分为：

| 板块 | 定位 | 代表企业 |
|------|------|----------|
| 主板（Main Board）| 成熟大型企业 | 腾讯(0700)、汇丰(0005)、阿里(9988) |
| 创业板（GEM）| 中小成长企业 | 规模相对较小 |

**港股的特殊性：**
- 全球国际化最高的中国企业上市地之一
- 大量内地红筹股、H股在港上市（如中国移动、中国建设银行）
- 同时有外资蓝筹（如汇丰、渣打）

### 2. 股票代码规则

```
港股代码为4~5位数字：
0700  腾讯控股
9988  阿里巴巴（同股不同权）
0005  汇丰控股
2318  中国平安（H股）
```

**H股 vs 红筹股 vs P股：**
- **H股**：在内地注册、在香港上市（如中石油H股 0857）
- **红筹股**：注册地在香港/海外，但主营业务在内地（如中国移动 0941）
- **P股**：注册地在内地以外（如开曼），同时有A、H两地上市

### 3. 核心交易规则

**交易时间（香港时间，=北京时间）：**
```
开市前时段（竞价）：09:00 - 09:30
持续交易时段：09:30 - 12:00 / 13:00 - 16:00
收市竞价时段：16:00 - 16:10
```

**与 A股的核心差异：**

| 对比项 | A股 | 港股 |
|--------|-----|------|
| 交易单位 | 1手=100股（固定）| 手数因股而异（10/100/500/1000股）|
| 涨跌停 | ±10% / ±20% | **无涨跌停** |
| 交收制度 | T+1（买入次日可卖）| **T+2** 交收 |
| 交易货币 | 人民币 | 港币（HKD）|
| 卖空机制 | 严格限制 | 可进行卖空（有指定名单）|
| 衍生品 | 有限 | 丰富（窝轮、牛熊证等）|

**费用结构：**
- 交易费：0.005%
- 证监会征费：0.0027%
- 联交所交易征费：0.00565%
- 印花税：0.13%（**买卖双向**，高于A股）
- 最低手续费：因券商而异

### 4. 指数体系

| 指数 | 成分股数量 | 特点 |
|------|-----------|------|
| 恒生指数（HSI）| 约80只 | 最权威的港股蓝筹指数，2021年大扩容 |
| 国企指数（HSCEI）| 约50只 | 在港上市的国有企业集合 |
| 恒生科技指数 | 30只 | 2020年推出，覆盖腾讯、美团、阿里等科技龙头 |
| 恒生综合指数 | 500只 | 更广泛的市场代表 |

### 5. 沪深港通机制

**什么是沪深港通？**

2014年沪港通、2016年深港通开通，打通了内地与香港资本市场的双向投资渠道。

```
北向资金（沪股通/深股通）：
  → 香港投资者/外资 购买 A 股
  → 被视为"聪明钱"的代表

南向资金（港股通）：
  → 内地投资者 购买 港股
  → 每日额度：500亿元人民币（净买入额度）
```

**港股通标的限制：**
- 恒生综合大型股指数 + 中型股指数成分股
- 同时在香港联交所和内地证交所上市的A+H股
- 股价须高于 0.01 港元

## 💡 示例 / 实操

### 使用 akshare 获取港股数据

```python
import akshare as ak
import pandas as pd

# 获取恒生指数历史数据
df_hsi = ak.stock_hk_index_daily_em(symbol="恒生指数")
print(df_hsi.tail())
```

### 获取腾讯控股（0700）日K数据

```python
# 获取港股个股数据（通过 akshare）
df_tencent = ak.stock_hk_daily(symbol="00700", adjust="qfq")
print(df_tencent.tail())
print(f"最新收盘价：{df_tencent['close'].iloc[-1]} HKD")
```

### 查看港股通南向资金数据

```python
# 获取南向资金历史流向
df_south = ak.stock_connect_south_hist_em()
print(df_south.tail())
```

## 📝 小结

**今天学到了什么：**

1. **港股是国际化程度最高的中国资产交易市场**，内地企业通过H股/红筹方式在港上市
2. **港股无涨跌停限制**，波动可以很极端（游资、窝轮助推）
3. **手数因股而异**，买入前必须确认每手股数，避免资金误判
4. **沪深港通是关键机制**，北向资金被视为外资指标，南向资金反映内地资金偏好
5. **AH溢价**是理解两地市场定价差异的重要视角

**关键术语：**
- H股：大陆注册、香港上市
- 红筹：香港/海外注册、主营在大陆
- T+2：港股交割周期，比A股的T+1多一天
- 窝轮/牛熊证：港股特有的杠杆衍生产品，风险极高
- AH溢价率：A股相对于H股的价格溢价程度
