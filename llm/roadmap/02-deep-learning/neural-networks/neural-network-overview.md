# 神经网络基础

## 1. 引言

深度学习是机器学习的一个分支，其核心是**人工神经网络**（Artificial Neural Networks, ANNs）。神经网络受到生物神经网络（尤其是大脑）结构和功能的启发，旨在模拟神经元之间信息传递和处理的方式。

对于一个从传统编程（如Java）转型的工程师来说，可以将神经网络想象成一个高度灵活且可学习的“函数”。我们不再手动编写复杂的if-else逻辑来解决问题，而是设计一个网络结构，然后通过大量的数据来“训练”它，让它自己学习到输入和输出之间的复杂映射关系。

例如，在图像识别任务中，我们不再手动编写规则来识别猫的耳朵、眼睛、胡须等特征，而是将猫的图片像素作为输入，将“猫”这个标签作为期望输出，让神经网络自动学习从像素到标签的映射。

## 2. 生物神经元与人工神经元

### 2.1 生物神经元（简化模型）

一个生物神经元主要由以下部分组成：
*   **细胞体（Cell Body/Soma）：** 处理信息。
*   **树突（Dendrites）：** 接收来自其他神经元的信号。
*   **轴突（Axon）：** 将信号传递给其他神经元。
*   **突触（Synapses）：** 神经元之间连接的点，信号通过突触传递。

工作方式：树突接收多个输入信号，细胞体对这些信号进行整合。如果整合后的信号强度超过某个阈值，神经元就会被激活，并通过轴突向下游神经元发送一个脉冲信号（动作电位）。

### 2.2 人工神经元（感知机模型 Perceptron）

人工神经元是对生物神经元的数学抽象。

**结构：**
*   **输入（Inputs）：** `x₁, x₂, ..., xₙ`。对应生物神经元树突接收的信号。
*   **权重（Weights）：** `w₁, w₂, ..., wₙ`。对应生物突触的强度。权重越大，该输入对神经元输出的影响越大。
*   **偏置（Bias）：** `b`。提供一个额外的激活阈值，增加模型的灵活性。
*   **加权求和：** `z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b = Σᵢ wᵢxᵢ + b`
*   **激活函数（Activation Function）：** `a = f(z)`。对应生物神经元的激活过程。它决定神经元是否应该被激活以及激活的程度。

**数学表达：**
```
a = f(Σᵢ wᵢxᵢ + b)
```

**常见激活函数：**
*   **阶跃函数（Step Function）：** 最简单的形式，`f(z) = 1 if z ≥ 0 else 0`。但因其不连续、不可导，在现代神经网络中很少直接使用。
*   **Sigmoid函数：** `σ(z) = 1 / (1 + e^(-z))`。输出范围 (0, 1)，平滑可导，但容易产生梯度消失问题。
*   **Tanh函数：** `tanh(z) = (e^z - e^(-z)) / (e^z + e^(-z))`。输出范围 (-1, 1)，是Sigmoid的缩放平移版本。
*   **ReLU函数（Rectified Linear Unit）：** `ReLU(z) = max(0, z)`。计算简单，有效缓解梯度消失问题，是目前最常用的激活函数之一。

## 3. 神经网络结构

### 3.1 层（Layer）

神经网络由多个**层**（Layer）堆叠而成。

*   **输入层（Input Layer）：** 接收原始数据（如图像像素、文本向量）。神经元数量通常等于输入特征的维度。
*   **隐藏层（Hidden Layers）：** 位于输入层和输出层之间。可以有一个或多个隐藏层。每一层的神经元接收前一层的输出作为输入，并将其处理后的结果传递给下一层。隐藏层是网络进行特征提取和抽象的地方。
*   **输出层（Output Layer）：** 产生网络的最终预测结果。神经元数量取决于任务类型：
    *   二分类：1个神经元（输出0或1，或概率）。
    *   多分类（C类）：C个神经元（通常配合Softmax激活函数，输出各类别的概率）。
    *   回归：1个或多个神经元（输出连续值）。

### 3.2 全连接层（Dense Layer / Fully Connected Layer）

在**全连接层**中，本层的每个神经元都与前一层的所有神经元相连。

**例子：** 假设第 `l-1` 层有 `n_{l-1}` 个神经元，第 `l` 层有 `n_l` 个神经元。
*   第 `l` 层的第 `j` 个神经元的输入为：`z_j^{[l]} = Σᵢ (w_{ij}^{[l]} * a_i^{[l-1]}) + b_j^{[l]}`
*   其输出（激活）为：`a_j^{[l]} = f(z_j^{[l]})`
*   其中 `w_{ij}^{[l]}` 是从第 `l-1` 层第 `i` 个神经元到第 `l` 层第 `j` 个神经元的权重，`b_j^{[l]}` 是第 `l` 层第 `j` 个神经元的偏置。

**向量化表示：**
*   令 `a^{[l-1]}` 为第 `l-1` 层的激活输出向量（维度 `n_{l-1} × 1`）。
*   令 `W^{[l]}` 为第 `l` 层的权重矩阵（维度 `n_l × n_{l-1}`）。
*   令 `b^{[l]}` 为第 `l` 层的偏置向量（维度 `n_l × 1`）。
*   则第 `l` 层的加权输入为：`z^{[l]} = W^{[l]} * a^{[l-1]} + b^{[l]}` （维度 `n_l × 1`）
*   第 `l` 层的激活输出为：`a^{[l]} = f(z^{[l]})` （维度 `n_l × 1`，`f` 作用在向量的每个元素上）

### 3.3 前向传播（Forward Propagation）

前向传播是指数据从输入层经过隐藏层，最终流向输出层的计算过程。

**过程：**
1.  输入数据 `X` 进入输入层。
2.  对于网络中的每一层 `l`：
    a. 计算加权输入：`z^{[l]} = W^{[l]} * a^{[l-1]} + b^{[l]}` （第一层 `a^{[0]}` 即为输入 `X`）
    b. 计算激活输出：`a^{[l]} = f^{[l]}(z^{[l]})`
3.  最终得到输出层的激活输出 `a^{[L]}`（`L` 为总层数），这就是网络的预测结果。

## 4. 损失函数（Loss Function / Cost Function）

前向传播给出了预测值，但我们如何知道这个预测有多好呢？**损失函数**用来衡量网络预测值 `ŷ` 与真实值 `y` 之间的差距。

训练的目标是最小化损失函数。

**常见的损失函数：**

1.  **均方误差（Mean Squared Error, MSE）：** 用于回归任务。
    ```
    J = (1/2m) * Σᵢ (ŷᵢ - yᵢ)²
    ```
    （`m` 是样本数量，`1/2` 为了求导方便）

2.  **二元交叉熵损失（Binary Cross-Entropy Loss）：** 用于二分类任务（输出层通常用Sigmoid激活）。
    ```
    J = -(1/m) * Σᵢ [yᵢ * log(ŷᵢ) + (1-yᵢ) * log(1-ŷᵢ)]
    ```

3.  **多元交叉熵损失（Categorical Cross-Entropy Loss）：** 用于多分类任务（输出层通常用Softmax激活）。
    ```
    J = -(1/m) * Σᵢ Σⱼ [y_{ij} * log(ŷ_{ij})]
    ```
    （其中 `y_{ij}` 是样本 `i` 属于类别 `j` 的真实标签，通常是one-hot编码）

## 5. 训练：梯度下降与反向传播（概览）

要训练神经网络，我们需要调整权重 `W` 和偏置 `b` 以最小化损失函数 `J`。这通常通过**梯度下降**（Gradient Descent）算法实现。

**核心思想：**
*   计算损失函数 `J` 对每个参数（`W` 和 `b`）的偏导数（梯度）。
*   沿着梯度的反方向更新参数，因为这是函数值下降最快的方向。
*   **更新规则：**
    ```
    W := W - α * (∂J/∂W)
    b := b - α * (∂J/∂b)
    ```
    其中 `α` 是学习率（Learning Rate），控制每次更新的步长。

**如何计算梯度？**
这就是**反向传播**（Backpropagation）算法要解决的问题。它是一种高效计算复合函数（神经网络）梯度的方法，本质上是链式法则的递归应用。我们将在后续章节详细讨论。

## 6. 总结

*   **神经网络**是受生物神经网络启发的计算模型，由人工神经元组成。
*   **人工神经元**接收输入，进行加权求和，加上偏置，然后通过激活函数产生输出。
*   神经网络通过**层**（输入层、隐藏层、输出层）堆叠而成，层与层之间通过权重和偏置连接。
*   **前向传播**是数据从输入到输出的计算过程。
*   **损失函数**衡量预测值与真实值的差距。
*   **训练**的目标是通过调整网络参数（权重和偏置）来最小化损失函数，主要方法是**梯度下降**，其梯度计算依赖于**反向传播**算法。

理解这些基本概念是学习更复杂网络结构（如CNN、RNN、Transformer）和训练技巧（如优化器、正则化）的基础。
