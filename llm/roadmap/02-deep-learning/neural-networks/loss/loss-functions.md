# 损失函数

## 1. 引言

在前向传播过程中，神经网络接收输入数据并产生预测输出。为了训练神经网络，使其预测结果尽可能接近真实标签，我们需要一个衡量预测值与真实值之间差异的指标。这个指标就是**损失函数**（Loss Function），也称为**代价函数**（Cost Function）。

损失函数是训练神经网络的核心。它定义了模型预测的“好坏”程度。训练的目标就是**最小化**这个损失函数，即找到一组最优的网络参数（权重和偏置），使得网络在训练数据上的平均损失尽可能小。

从优化的角度看，训练神经网络就是一个**优化问题**：在参数空间中寻找能使损失函数值最小的参数点。

## 2. 损失函数的基本概念

### 2.1 定义

对于一个训练样本 `i`，其输入为 `x^{(i)}`，真实标签为 `y^{(i)}`，神经网络的预测输出为 `ŷ^{(i)}`（也记作 `a^{[L](i)}`，表示第 `L` 层的输出）。**损失函数** `L` 衡量的是单个样本的预测值 `ŷ^{(i)}` 与真实值 `y^{(i)}` 之间的差异：

```
L^{(i)} = L(ŷ^{(i)}, y^{(i)})
```

对于包含 `m` 个样本的整个训练集，**代价函数**（Cost Function） `J` 是所有训练样本损失的平均值：

```
J = (1/m) * Σ (i=1 to m) L^{(i)} = (1/m) * Σ (i=1 to m) L(ŷ^{(i)}, y^{(i)})
```

在实际应用中，这两个术语经常混用，但严格来说，损失函数是针对单个样本的，而代价函数是针对整个数据集的。

### 2.2 损失函数的性质

一个好的损失函数通常需要满足以下性质：

1.  **非负性：** `L(ŷ, y) ≥ 0`。损失值非负，当且仅当预测完全正确时为0。
2.  **可微性：** 为了能够使用基于梯度的优化算法（如梯度下降），损失函数需要是可微的（至少在大部分区域可微）。
3.  **对预测误差敏感：** 当预测值 `ŷ` 偏离真实值 `y` 越远时，损失值应该越大。

## 3. 常见的损失函数

损失函数的选择高度依赖于具体的任务类型。

### 3.1 回归任务

回归任务的目标是预测一个连续值。

#### 均方误差（Mean Squared Error, MSE）

这是回归任务中最常用的损失函数。

**定义：**
```
L(ŷ, y) = (1/2) * (ŷ - y)²
J = (1/m) * Σ (i=1 to m) (1/2) * (ŷ^{(i)} - y^{(i)})²
```
（`1/2` 是为了在求导时抵消平方的系数，简化计算）

**性质：**
*   对较大的误差给予更高的惩罚（平方效应）。
*   处处可微。
*   对异常值（Outliers）敏感。

#### 平均绝对误差（Mean Absolute Error, MAE）

**定义：**
```
L(ŷ, y) = |ŷ - y|
J = (1/m) * Σ (i=1 to m) |ŷ^{(i)} - y^{(i)}|
```

**性质：**
*   对误差的惩罚是线性的。
*   对异常值的鲁棒性比MSE强。
*   在 `ŷ = y` 处不可微，但通常约定其导数为0。

#### Huber损失

Huber损失结合了MSE和MAE的优点，对小误差采用平方惩罚，对大误差采用线性惩罚，具有更强的鲁棒性。

**定义：**
```
L(ŷ, y) = {
    (1/2) * (ŷ - y)²,                           if |ŷ - y| ≤ δ
    δ * |ŷ - y| - (1/2) * δ²,                  if |ŷ - y| > δ
}
```
其中 `δ` 是一个超参数，用于确定从平方损失到线性损失的转换点。

### 3.2 分类任务

分类任务的目标是预测样本属于哪个类别。

#### 二元分类（Binary Classification）

通常输出层只有一个神经元，激活函数为Sigmoid，输出 `ŷ` 表示属于正类（类别1）的概率。

**二元交叉熵损失（Binary Cross-Entropy Loss）**

**定义：**
```
L(ŷ, y) = - [y * log(ŷ) + (1 - y) * log(1 - ŷ)]
J = -(1/m) * Σ (i=1 to m) [y^{(i)} * log(ŷ^{(i)}) + (1 - y^{(i)}) * log(1 - ŷ^{(i)})]
```
其中 `y ∈ {0, 1}` 是真实标签。

**性质：**
*   当真实标签 `y=1` 时，损失鼓励 `ŷ` 接近1；当 `y=0` 时，损失鼓励 `ŷ` 接近0。
*   当预测错误且置信度高时（例如 `y=1` 但 `ŷ` 接近0），损失会变得非常大。
*   当预测正确且置信度高时（例如 `y=1` 且 `ŷ` 接近1），损失会接近0。
*   处处可微（在 `ŷ=0` 或 `ŷ=1` 时需做数值处理以避免 `log(0)`）。

#### 多元分类（Multiclass Classification）

通常输出层有 `C` 个神经元（`C` 是类别数），激活函数为Softmax，输出 `ŷ` 是一个C维概率分布向量。

**类别标签表示：**
*   **硬标签（Hard Labels）：** 使用**独热编码**（One-Hot Encoding）。例如，对于3分类问题，如果样本属于类别2，则标签 `y = [0, 1, 0]ᵀ`。
*   **软标签（Soft Labels）：** 标签本身就是一个概率分布，常用于标签平滑（Label Smoothing）等技术。

**多元交叉熵损失（Categorical Cross-Entropy Loss）**

**定义：**
```
L(ŷ, y) = - Σ (c=1 to C) y_c * log(ŷ_c)
J = -(1/m) * Σ (i=1 to m) Σ (c=1 to C) y_c^{(i)} * log(ŷ_c^{(i)})
```
其中 `y_c` 是样本属于类别 `c` 的真实概率（对于硬标签，只有一个 `y_c=1`，其余为0）。

**性质：**
*   鼓励预测概率分布 `ŷ` 接近真实概率分布 `y`。
*   处处可微（需避免 `log(0)`）。

#### 稀疏分类（Sparse Classification）

这是多元分类的一种特殊情况。真实标签 `y` 直接用类别索引表示（如 `y=2` 表示属于类别2），而不是使用独热编码。这在实现上更节省内存。

损失函数的数学形式与多元交叉熵相同，只是在计算时直接使用 `y` 作为索引：
```
L(ŷ, y) = - log(ŷ_y)
```

## 4. 损失函数与激活函数的搭配

损失函数的选择通常与输出层的激活函数紧密相关，以确保整体的数学性质良好（如凸性）和优化效率。

*   **回归（输出无界）：**
    *   激活函数：通常无（恒等函数）或Tanh。
    *   损失函数：MSE, MAE。
*   **二元分类（输出概率）：**
    *   激活函数：Sigmoid。
    *   损失函数：二元交叉熵。
*   **多元分类（输出概率分布）：**
    *   激活函数：Softmax。
    *   损失函数：多元交叉熵。

这种搭配使得损失函数关于最后一层的加权输入 `z^{[L]}` 是**凸函数**，有助于优化算法找到全局最优解（对于单层输出而言）。

## 5. 损失函数在训练中的作用

1.  **衡量性能：** 提供一个量化的指标来评估模型在训练集（和验证集）上的表现。
2.  **驱动优化：** 损失函数的梯度（通过反向传播计算）是更新网络参数的依据。梯度下降法沿着负梯度方向更新参数，以最小化损失。
3.  **模型选择：** 不同的损失函数可能导致模型学习到不同的特征。例如，MAE比MSE更关注中位数而非均值。

## 6. 总结

*   **损失函数**是衡量模型预测值与真实值差异的指标，是训练过程的核心目标。
*   **代价函数**是整个训练集上损失函数的平均值。
*   损失函数的选择依赖于任务类型：
    *   **回归：** MSE, MAE, Huber。
    *   **二元分类：** 二元交叉熵（搭配Sigmoid）。
    *   **多元分类：** 多元交叉熵（搭配Softmax）。
*   损失函数需要是可微的，以便使用梯度下降等优化算法。
*   理解损失函数对于理解模型训练目标、调试训练过程和选择合适的模型架构至关重要。
