# 优化器基础

## 1. 引言

在深度学习中，训练模型的核心是通过优化算法来调整模型参数，以最小化损失函数。在之前的章节中，我们学习了**梯度下降**（Gradient Descent）及其变种（SGD, Mini-batch GD）和核心的**反向传播**算法。梯度下降是优化的基础，但它存在一些局限性，例如：

*   **学习率选择困难：** 固定的学习率可能在初期过大导致震荡，后期过小导致收敛缓慢。
*   **收敛速度慢：** 在某些情况下（如损失函数的等高线呈狭长的椭圆形），梯度下降可能会来回震荡，收敛到最优解的路径很长。
*   **容易陷入局部最优或鞍点：** 特别是在高维非凸优化问题中。

为了解决这些问题，研究者们提出了许多**优化器**（Optimizer），它们在标准梯度下降的基础上引入了各种改进策略，以实现更快、更稳定、更智能的参数更新。

## 2. 经典优化器

### 2.1 随机梯度下降（Stochastic Gradient Descent, SGD）

这是梯度下降最基本的变种。

**更新规则：**
```
θ = θ - α * ∇J(θ; x^{(i)}, y^{(i)})
```
其中 `(x^{(i)}, y^{(i)})` 是一个随机选取的训练样本。

**优点：**
*   更新频繁，可以跳出局部最优。
*   计算开销小。

**缺点：**
*   梯度波动大，收敛不稳定。
*   难以选择合适的学习率。

### 2.2 动量法（Momentum）

SGD的梯度方向波动很大，导致收敛过程曲折。**动量法**的灵感来源于物理学，它模拟了物体在运动中积累的“动量”。

**核心思想：**
不仅仅基于当前的梯度更新参数，还考虑之前更新的“惯性”或“动量”。这有助于加速沿一致方向的移动，并抑制震荡。

**更新规则：**
```
v = β * v + (1 - β) * ∇J(θ)  // 更新速度（动量）
θ = θ - α * v                // 更新参数
```
*   `v` 是速度向量（与参数 `θ` 维度相同），初始为0。
*   `β` 是动量因子（Momentum Term），通常取0.9左右。它决定了之前速度的保留比例。
*   `∇J(θ)` 是当前梯度。
*   `(1 - β)` 是当前梯度的权重。

**解释：**
*   `v` 可以看作是梯度的指数移动平均（Exponentially Weighted Average）。
*   如果梯度方向持续一致，`v` 会不断累积，使得更新步长越来越大，加速收敛。
*   如果梯度方向来回震荡，正负梯度会相互抵消，`v` 趋于平缓，从而减少震荡。

### 2.3 Nesterov加速梯度（Nesterov Accelerated Gradient, NAG）

NAG是对标准动量法的一个改进。它不是在当前位置计算梯度，而是先根据上一步的动量做一个“前瞻”（look-ahead），在前瞻的位置计算梯度，然后再进行更新。

**更新规则：**
```
θ_lookahead = θ + β * v
v = β * v + (1 - β) * ∇J(θ_lookahead)
θ = θ - α * v
```

**优点：**
*   比标准动量法有更好的收敛性，能更有效地减少震荡。

## 3. 自适应学习率优化器

这类优化器的核心思想是为每个参数维护一个独立的学习率，并根据参数的历史梯度信息自动调整该学习率。

### 3.1 AdaGrad（Adaptive Gradient Algorithm）

AdaGrad根据参数的历史梯度的平方和来调整每个参数的学习率。对于历史梯度平方和较大的参数，其学习率会减小；反之则增大。

**更新规则：**
```
s = s + (∇J(θ))²          // 累积梯度平方和 (element-wise)
θ = θ - (α / √(s + ε)) * ∇J(θ) // 更新参数
```
*   `s` 是累积梯度平方和向量（与 `θ` 维度相同），初始为0。
*   `ε` 是一个极小的常数（如 `1e-8`），防止除零错误。
*   `√(s + ε)` 是对向量 `s + ε` 的每个元素开平方根（element-wise）。

**优点：**
*   对于稀疏数据（如自然语言处理中的词嵌入），能对不频繁出现的特征给予更大的更新。

**缺点：**
*   `s` 会持续累积增大，导致学习率单调递减，最终可能变得过小，训练提前停止。

### 3.2 RMSprop（Root Mean Square Propagation）

RMSprop是为了解决AdaGrad学习率过快衰减的问题而提出的。它不像AdaGrad那样累积所有历史梯度的平方，而是使用**指数移动平均**来计算梯度平方的平均值。

**更新规则：**
```
s = β * s + (1 - β) * (∇J(θ))² // 梯度平方的指数移动平均
θ = θ - (α / √(s + ε)) * ∇J(θ)  // 更新参数
```
*   `β` 通常取0.9。

**解释：**
*   通过指数移动平均，远古的梯度信息会逐渐被遗忘，使得学习率不会单调递减至零。
*   RMSprop在非稳态（non-stationary）目标函数上表现良好。

### 3.3 Adam（Adaptive Moment Estimation）

Adam是目前最流行和广泛使用的优化器之一。它结合了**动量法**（Momentum）和**RMSprop**的优点。

**核心思想：**
*   使用梯度的一阶矩估计（指数移动平均）来计算动量（类似Momentum）。
*   使用梯度的二阶矩估计（指数移动平均的平方）来调整学习率（类似RMSprop）。

**更新规则：**
```
m = β₁ * m + (1 - β₁) * ∇J(θ)       // 一阶矩估计 (动量)
s = β₂ * s + (1 - β₂) * (∇J(θ))²    // 二阶矩估计 (梯度平方的指数移动平均)
m_corrected = m / (1 - β₁^t)        // 偏差修正
s_corrected = s / (1 - β₂^t)        // 偏差修正
θ = θ - α * m_corrected / (√s_corrected + ε) // 更新参数
```
*   `m` 是动量向量，初始为0。
*   `s` 是梯度平方的指数移动平均向量，初始为0。
*   `β₁` 是一阶矩估计的衰减率，常用值为0.9。
*   `β₂` 是二阶矩估计的衰减率，常用值为0.999。
*   `t` 是当前的迭代步数（epoch或step）。
*   `m_corrected` 和 `s_corrected` 是偏差修正项，因为 `m` 和 `s` 初始为0，导致在训练初期估计有偏差。

**优点：**
*   计算效率高。
*   内存需求少。
*   对梯度稀疏或噪声较多的情况鲁棒。
*   超参数（`α`, `β₁`, `β₂`）通常有较好的默认值，对超参数调整不敏感。

## 4. 其他优化器简介

*   **AdaDelta：** AdaGrad的改进版，解决了学习率单调衰减的问题，且不需要设置初始学习率。
*   **Adamax：** Adam的变种，使用无穷范数（max norm）代替二阶矩估计。
*   **Nadam：** Nesterov动量版的Adam。
*   **AdaBound：** 将Adam的自适应学习率逐渐限制在一个固定的范围内，结合了Adam的快速收敛和SGD的最终性能。

## 5. 如何选择优化器？

*   **Adam** 是一个非常好的默认选择，适用于大多数情况。它结合了动量和自适应学习率的优点，通常收敛快且稳定。
*   **SGD with Momentum** 在某些情况下（尤其是训练后期）可能达到比Adam更好的泛化性能，但需要仔细调整学习率。
*   **RMSprop** 适用于非稳态目标函数。
*   最终选择哪个优化器，以及如何设置超参数，往往需要在具体任务和数据集上进行实验比较。

## 6. 学习率调度（Learning Rate Scheduling）

除了选择优化器，**动态调整学习率**也是提升训练效果的重要手段。

**常见策略：**
*   **Step Decay：** 训练过程中，每隔几个epoch将学习率乘以一个衰减因子（如0.1）。
*   **Exponential Decay：** 学习率按指数函数衰减。
*   **Cosine Annealing：** 学习率按余弦函数变化，从初始值逐渐减小到接近0。
*   **ReduceLROnPlateau：** 当监测到损失（或指标）停止改善时，降低学习率。

## 7. 总结

*   **优化器**是深度学习训练的核心组件，用于更新模型参数以最小化损失函数。
*   从基础的**SGD**，到引入**动量**（Momentum, NAG）以加速收敛和减少震荡。
*   再到**自适应学习率**优化器（AdaGrad, RMSprop, Adam），它们能为不同参数动态调整学习率，提高训练效率和稳定性。
*   **Adam**因其良好的综合性能而成为最广泛使用的优化器。
*   结合**学习率调度**策略，可以进一步优化训练过程。

理解各种优化器的原理和特点，有助于在实际项目中选择合适的优化策略，提高模型训练的效率和最终性能。
