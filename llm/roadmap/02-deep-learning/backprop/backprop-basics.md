# 梯度与反向传播基础

## 1. 引言

在神经网络基础中，我们了解到训练神经网络的目标是通过调整网络的参数（权重 `W` 和偏置 `b`）来最小化损失函数 `J`。这是一个典型的**优化问题**。

**梯度下降**（Gradient Descent）是解决这一优化问题的核心算法。其基本思想是：
*   计算损失函数 `J` 相对于每个参数的**梯度**（偏导数）。
*   沿着梯度的**反方向**更新参数，因为这是函数值**下降最快**的方向。

然而，对于包含数百万甚至数十亿参数的深度神经网络，手动计算每个参数的梯度是不现实的。**反向传播**（Backpropagation, BP）算法应运而生，它提供了一种高效、系统化地计算神经网络中所有参数梯度的方法。

## 2. 梯度（Gradient）

### 2.1 定义

对于一个多元函数 `f(x₁, x₂, ..., xₙ)`，其**梯度**是一个向量，包含函数对每个自变量的偏导数：

```
∇f = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]ᵀ
```

在神经网络中，损失函数 `J` 是所有参数（`W` 和 `b`）的函数。梯度 `∇J` 指向了 `J` 增加最快的方向，因此 `-∇J` 就指向了 `J` 减少最快的方向。

### 2.2 梯度下降算法

梯度下降的更新规则为：

```
θ := θ - α * ∇J(θ)
```

其中：
*   `θ` 代表所有待优化的参数（可以是 `W` 或 `b`）。
*   `α` 是**学习率**（Learning Rate），控制每次更新的步长。
*   `∇J(θ)` 是损失函数 `J` 在参数 `θ` 处的梯度。

**学习率 `α` 的选择：**
*   `α` 太大：更新步长过大，可能导致越过最优解，甚至使损失函数值震荡或发散。
*   `α` 太小：收敛速度过慢。

## 3. 反向传播算法（Backpropagation）

反向传播是计算神经网络梯度的高效算法。它本质上是**链式法则**（Chain Rule）在计算图上的递归应用。

### 3.1 计算图（Computational Graph）

神经网络的前向传播过程可以表示为一个**有向无环图**（DAG），称为计算图。图中的节点代表变量（输入、参数、中间变量、输出），边代表变量间的函数依赖关系。

**例子：** 对于简单的函数 `J = (x + y) * z`，其中 `x, y, z` 是输入。
```
  x ----\
        --> u = x + y --> v = u * z --> J
  y ----/               /
                    z -/
```

### 3.2 链式法则回顾

如果 `z = f(x, y)` 且 `x = g(t)`, `y = h(t)`，那么：
```
dz/dt = (∂z/∂x) * (dx/dt) + (∂z/∂y) * (dy/dt)
```

在计算图中，这对应于从输出节点到输入节点的所有路径的偏导数乘积之和。

### 3.3 反向传播过程

反向传播分为两个阶段：

#### 前向传播（Forward Pass）
*   从输入层开始，按照网络结构逐层计算，直到得到最终的损失值 `J`。
*   在此过程中，需要保存每一层的中间计算结果（如 `z`, `a`），因为它们在反向传播时需要用到。

#### 反向传播（Backward Pass）
*   从输出层（损失 `J`）开始，计算 `J` 对最后一层输出的梯度。
*   然后，利用链式法则，逐层向后（从输出层向输入层）计算损失 `J` 对每一层参数和激活的梯度。
*   具体来说，对于每一层 `l`：
    1.  计算 `J` 对该层加权输入 `z^{[l]}` 的梯度：`dZ^{[l]} = dA^{[l]} * g'(Z^{[l]})` （其中 `g` 是激活函数）。
    2.  计算 `J` 对该层权重 `W^{[l]}` 和偏置 `b^{[l]}` 的梯度：
        *   `dW^{[l]} = (1/m) * dZ^{[l]} * A^{[l-1]T}` （`m` 是样本数）
        *   `db^{[l]} = (1/m) * sum(dZ^{[l]}, axis=1, keepdims=True)`
    3.  计算传递给前一层的梯度 `dA^{[l-1]}`：
        *   `dA^{[l-1]} = W^{[l]T} * dZ^{[l]}`

### 3.4 一个简单的例子

考虑一个单层神经网络：`z = W * x + b`，`a = σ(z)`，损失 `J = (1/2)(a - y)²`。

**前向传播：**
1.  `z = W * x + b`
2.  `a = σ(z)`
3.  `J = (1/2)(a - y)²`

**反向传播：**
1.  计算 `J` 对 `a` 的梯度：`dJ/da = a - y`
2.  计算 `J` 对 `z` 的梯度：`dJ/dz = (dJ/da) * σ'(z) = (a - y) * σ'(z)`
3.  计算 `J` 对 `W` 和 `b` 的梯度：
    *   `dJ/dW = (dJ/dz) * x`
    *   `dJ/db = dJ/dz`

## 4. 反向传播在深度网络中的应用

对于深度网络，反向传播的过程是上述过程的重复和扩展。

**关键点：**
1.  **递归性：** 每一层的梯度计算都依赖于后一层传递回来的梯度。
2.  **高效性：** 通过一次前向传播和一次反向传播，可以同时计算出所有参数的梯度，避免了对每个参数单独求导的低效过程。
3.  **实现：** 现代深度学习框架（如TensorFlow, PyTorch）都内置了自动微分（Automatic Differentiation）功能，可以自动执行反向传播，开发者通常不需要手动实现。

## 5. 梯度下降的变种

基础的梯度下降在实际应用中有一些变种，以提高效率和稳定性：

1.  **批量梯度下降（Batch Gradient Descent）：** 使用整个训练集计算梯度。优点是梯度稳定，缺点是计算量大，内存需求高。
2.  **随机梯度下降（Stochastic Gradient Descent, SGD）：** 每次只使用一个训练样本计算梯度并更新参数。优点是更新频繁，可以跳出局部最优；缺点是梯度波动大，收敛不稳定。
3.  **小批量梯度下降（Mini-batch Gradient Descent）：** 每次使用一小批（Mini-batch）样本计算梯度。这是目前最常用的方法，平衡了稳定性和效率。

## 6. 总结

*   **梯度**是多元函数增长最快方向的向量，梯度的反方向是函数下降最快的方向。
*   **梯度下降**算法通过沿着负梯度方向更新参数来最小化损失函数。
*   **反向传播**是计算神经网络梯度的高效算法，它是链式法则在计算图上的系统化应用。
*   反向传播分为**前向传播**（计算损失）和**反向传播**（计算梯度）两个阶段。
*   现代深度学习框架通过自动微分实现了高效的反向传播。
*   梯度下降有多种变种（SGD, Mini-batch GD），用于在不同场景下平衡计算效率和收敛稳定性。

理解梯度和反向传播是掌握深度学习训练过程的关键。
