# RNN基础

## 1. 引言

在前面学习的全连接网络和CNN中，我们处理的数据（如图像）通常是**固定大小**且**元素之间没有明显的顺序依赖**。然而，现实世界中存在着大量**序列数据**（Sequential Data），其中元素的顺序至关重要，并且前后元素之间存在依赖关系。

**序列数据的例子：**
*   **自然语言：** 一个句子中的词。"我爱"之后很可能是"编程"而不是"面包"。
*   **语音信号：** 音频波形随时间变化。
*   **时间序列：** 股票价格、气温、点击率等随时间变化的数据。
*   **视频：** 一系列按时间排序的图像帧。

传统的神经网络难以有效处理这类数据，因为它们：
1.  **输入/输出尺寸固定：** 无法处理变长序列。
2.  **缺乏记忆：** 无法利用序列中先前元素的信息来影响对当前元素的处理。

**循环神经网络**（Recurrent Neural Network, RNN）就是为了处理序列数据而设计的。它的核心思想是**网络中存在环路**，使得信息可以在网络中循环流动，从而具备了**记忆**先前信息的能力。

## 2. RNN的核心思想与结构

### 2.1 基本结构

RNN的核心在于其**隐藏状态**（Hidden State）。在处理序列中的每个元素时，RNN不仅会考虑当前的输入，还会考虑上一时刻的隐藏状态。

**数学表示：**
对于一个输入序列 `x^{<1>}, x^{<2>}, ..., x^{<T>}` （`T` 是序列长度），RNN在每个时间步 `t` 的计算如下：

1.  **隐藏状态更新：**
    ```
    h^{<t>} = g(W_{hh} * h^{<t-1>} + W_{xh} * x^{<t>} + b_h)
    ```
    *   `h^{<t>}`：时间步 `t` 的隐藏状态（输出）。
    *   `h^{<t-1>`：时间步 `t-1` 的隐藏状态（输入）。
    *   `x^{<t>}`：时间步 `t` 的输入。
    *   `W_{hh}`：隐藏状态到隐藏状态的权重矩阵。
    *   `W_{xh}`：输入到隐藏状态的权重矩阵。
    *   `b_h`：隐藏状态的偏置向量。
    *   `g`：激活函数（常用tanh或ReLU）。

2.  **输出计算（可选）：**
    ```
    y^{<t>} = W_{hy} * h^{<t>} + b_y
    ```
    *   `y^{<t>}`：时间步 `t` 的输出。
    *   `W_{hy}`：隐藏状态到输出的权重矩阵。
    *   `b_y`：输出的偏置向量。

**初始状态：** 通常设定 `h^{<0>} = 0` 向量。

### 2.2 循环的直观理解

RNN可以被展开成一个链式结构，每个时间步都是一个相同的神经网络单元（Cell），它们共享相同的参数 `W_{hh}`, `W_{xh}`, `W_{hy}`, `b_h`, `b_y`。

```
h^{<0>} -> Cell -> h^{<1>} -> Cell -> h^{<2>} -> ... -> Cell -> h^{<T>}
           x^{<1>}           x^{<2>}                     x^{<T>}
           y^{<1>}           y^{<2>}                     y^{<T>}
```

这种结构使得RNN在处理当前时间步 `t` 的输入 `x^{<t>}` 时，能够通过隐藏状态 `h^{<t-1>}` 间接地“记住”之前所有时间步的信息。

### 2.3 不同类型的RNN

根据任务需求，RNN可以有不同的输入输出结构：

1.  **一对一（One-to-One）：** 传统神经网络，非RNN。
2.  **一对多（One-to-Many）：** 输入一个信号，输出一个序列。例如，图像描述生成（输入图像，输出描述句子）。
3.  **多对一（Many-to-One）：** 输入一个序列，输出一个信号。例如，情感分析（输入句子，输出情感类别）。
4.  **多对多（Many-to-Many 同步）：** 输入一个序列，输出一个等长序列。例如，词性标注（输入句子，输出每个词的词性）。
5.  **多对多（Many-to-Many 异步）：** 输入一个序列，输出另一个序列。例如，机器翻译（输入源语言句子，输出目标语言句子）。

## 3. RNN的应用

RNN及其变体（如LSTM、GRU）在处理序列数据方面取得了巨大成功，广泛应用于：

*   **自然语言处理（NLP）：**
    *   语言模型（预测下一个词）
    *   机器翻译
    *   文本生成
    *   命名实体识别
    *   情感分析
*   **语音识别：** 将音频波形序列转换为文本序列。
*   **时间序列预测：** 预测股票价格、天气等。
*   **音乐生成：** 生成旋律序列。

## 4. RNN的挑战：梯度消失与梯度爆炸

尽管RNN理论上可以捕获长距离依赖，但在实际训练中却面临严重问题。

### 4.1 问题根源

在反向传播（BPTT, Backpropagation Through Time）过程中，梯度需要通过链式法则从输出层向输入层传播。对于RNN，这意味着梯度需要通过 `W_{hh}` 矩阵多次相乘。

*   如果 `W_{hh}` 的特征值**大于1**，梯度在反向传播时会**指数级增长**，导致**梯度爆炸**（Gradient Explosion）。
*   如果 `W_{hh}` 的特征值**小于1**，梯度在反向传播时会**指数级衰减**，导致**梯度消失**（Gradient Vanishing）。

### 4.2 影响

*   **梯度爆炸：** 导致权重更新过大，训练不稳定，损失函数变为NaN。
*   **梯度消失：** 网络难以学习到长期依赖关系。例如，在句子“The cat, which already ate ..., was full”中，网络很难将“was full”与“cat”联系起来，因为它们之间隔着很多词。

### 4.3 解决方案（概览）

为了解决这些问题，研究者提出了更复杂的RNN单元：

*   **长短期记忆网络（LSTM, Long Short-Term Memory）：** 通过引入门控机制（输入门、遗忘门、输出门）来控制信息的流动，有效缓解梯度消失问题，是捕捉长距离依赖的常用方法。
*   **门控循环单元（GRU, Gated Recurrent Unit）：** LSTM的简化版本，将遗忘门和输入门合并为更新门，并省略了输出门，计算效率更高，性能与LSTM相当。

## 5. 总结

*   **RNN** 是一种专门用于处理序列数据的神经网络，其核心是具有**隐藏状态**和**循环连接**的神经元。
*   隐藏状态 `h^{<t>}` 允许网络在处理当前输入 `x^{<t>}` 时，利用序列中先前的信息。
*   RNN可以处理多种输入输出结构，如多对一、多对多等，适用于NLP、语音、时间序列分析等领域。
*   **梯度消失**和**梯度爆炸**是训练RNN时的主要困难，限制了其捕获长距离依赖的能力。
*   **LSTM**和**GRU**等改进的RNN单元通过引入门控机制，有效地缓解了梯度消失问题，成为处理长序列的主流方法。

虽然RNN及其变体在序列建模中曾占主导地位，但近年来基于**注意力机制**（Attention Mechanism）的**Transformer**架构因其并行化优势和更强的建模能力，逐渐成为新的主流。
