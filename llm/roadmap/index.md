# 📚 LLM Roadmap — 系统学习参考资料

> 内容来源：[llm-roadmap](https://github.com/shouldnotappearcalm/llm-roadmap)

## 🗺️ 推荐学习路径

```
扎实数学与编程基础
  → 深入理解 Transformer 架构
  → 熟练掌握微调（Fine-tuning）与 RAG 技术
  → 模型优化与部署实践
```

**具体建议路径：** Python + PyTorch 基础 → NLP 基础 → Transformer/LLM 理论 → HuggingFace 生态 → 提示词工程 → RAG 与 Agent 开发 → LoRA/QLoRA 微调 → 分布式训练与量化部署
