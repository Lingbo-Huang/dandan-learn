# 积分与基本定理基础

## 1. 引言

微积分学包含两个主要部分：**微分**（Differential Calculus）和**积分**（Integral Calculus）。我们已经学习了微分，它关注的是函数的变化率（导数）。积分则关注的是**累积**（Accumulation）或**总量**（Total Amount）的问题。

想象一个水龙头以变化的流速 `r(t)`（单位：升/分钟）放水，你想知道从时间 `t=a` 到 `t=b` 总共放了多少水。这个问题的答案就是对流速函数 `r(t)` 从 `a` 到 `b` 进行积分。积分可以看作是微分的逆运算。

## 2. 积分的概念

### 2.1 定积分（Definite Integral）

定积分用来计算函数在某个区间 `[a, b]` 上与 x 轴围成的**有向面积**（Signed Area）。

**定义（黎曼和的极限）：**

将区间 `[a, b]` 分成 `n` 个小区间，每个小区间的宽度为 `Δx = (b-a)/n`。在每个小区间 `[x_{i-1}, x_i]` 内任取一点 `x_i*`，构造一个矩形，其高度为 `f(x_i*)`，宽度为 `Δx`。所有这些矩形的面积之和称为**黎曼和**（Riemann Sum）：

```
S_n = Σ (i=1 to n) f(x_i*) * Δx
```

当分割越来越细（即 `n -> ∞`，`Δx -> 0`）时，如果这个黎曼和的极限存在，那么这个极限值就定义为函数 `f(x)` 在区间 `[a, b]` 上的**定积分**，记作：

```
∫[a, b] f(x) dx
```

*   `∫` 是积分符号。
*   `a` 是积分下限，`b` 是积分上限。
*   `f(x)` 是被积函数。
*   `dx` 表示积分变量是 `x`，`dx` 代表无穷小的宽度。

**几何意义：**
*   定积分 `∫[a, b] f(x) dx` 表示函数 `f(x)` 的图像与 `x` 轴以及直线 `x=a` 和 `x=b` 所围成的区域的**有向面积**。
*   如果 `f(x)` 在 `[a, b]` 上非负，那么定积分就是实际的面积。
*   如果 `f(x)` 在 `[a, b]` 上有正有负，那么定积分是各部分面积的代数和（x轴上方的面积减去x轴下方的面积）。

### 2.2 不定积分（Indefinite Integral） / 原函数（Antiderivative）

不定积分关注的不是某个区间上的累积量，而是寻找一个函数的**原函数**（Antiderivative）。

**定义：**

如果 `F'(x) = f(x)`，那么函数 `F(x)` 就称为 `f(x)` 的一个**原函数**。`f(x)` 的所有原函数构成的集合称为 `f(x)` 的**不定积分**，记作：

```
∫ f(x) dx = F(x) + C
```

*   `∫` 是积分符号。
*   `f(x)` 是被积函数。
*   `dx` 表示积分变量是 `x`。
*   `F(x)` 是 `f(x)` 的一个原函数。
*   `C` 是任意常数，称为**积分常数**。因为常数的导数为0，所以 `F(x) + C` 的导数仍然是 `f(x)`。

**例子：**
*   因为 `(x²)' = 2x`，所以 `∫ 2x dx = x² + C`。
*   因为 `(sin(x))' = cos(x)`，所以 `∫ cos(x) dx = sin(x) + C`。

## 3. 微积分基本定理（Fundamental Theorem of Calculus）

微积分基本定理是连接微分和积分这两个看似不同概念的桥梁，它表明了微分和积分互为逆运算。

### 3.1 第一部分（微积分第一基本定理）

如果函数 `f` 在区间 `[a, b]` 上连续，并且我们定义一个新的函数 `F(x)` 如下：

```
F(x) = ∫[a, x] f(t) dt
```

（这里 `t` 是积分变量，`x` 是上限）

那么，函数 `F(x)` 在 `[a, b]` 上可导，并且它的导数就是 `f(x)`：

```
F'(x) = d/dx [∫[a, x] f(t) dt] = f(x)
```

**解释：**
这个定理告诉我们，**变上限积分**（上限是变量 `x`）本身就是一个原函数。它建立了积分（累积过程）和微分（变化率）之间的直接联系。

### 3.2 第二部分（微积分第二基本定理 / 牛顿-莱布尼茨公式）

如果函数 `f` 在区间 `[a, b]` 上连续，且 `F` 是 `f` 的一个原函数（即 `F'(x) = f(x)`），那么：

```
∫[a, b] f(x) dx = F(b) - F(a)
```

这个公式也常记作 `F(x)|[a, b]`。

**解释：**
这个定理提供了一种计算定积分的简便方法。我们不需要通过计算复杂的黎曼和的极限，只需要找到被积函数 `f(x)` 的一个原函数 `F(x)`，然后计算 `F` 在上限和下限处的值的差即可。

**例子：** 计算 `∫[0, 2] x² dx`。

*   找到 `x²` 的一个原函数：`F(x) = x³/3` （因为 `(x³/3)' = x²`）。
*   应用牛顿-莱布尼茨公式：
    ```
    ∫[0, 2] x² dx = F(2) - F(0) = (2³/3) - (0³/3) = 8/3 - 0 = 8/3
    ```

## 4. 基本积分公式与法则

为了方便计算积分，我们需要记住一些基本的积分公式和法则，它们大多可以通过求导公式反推得到。

### 4.1 基本积分公式

*   `∫ k dx = kx + C` （`k` 是常数）
*   `∫ xⁿ dx = x^(n+1) / (n+1) + C` （`n ≠ -1`）
*   `∫ 1/x dx = ln|x| + C`
*   `∫ e^x dx = e^x + C`
*   `∫ a^x dx = a^x / ln(a) + C` （`a > 0, a ≠ 1`）
*   `∫ sin(x) dx = -cos(x) + C`
*   `∫ cos(x) dx = sin(x) + C`
*   `∫ sec²(x) dx = tan(x) + C`
*   `∫ csc²(x) dx = -cot(x) + C`
*   `∫ sec(x)tan(x) dx = sec(x) + C`
*   `∫ csc(x)cot(x) dx = -csc(x) + C`

### 4.2 积分法则

1.  **线性性（Linearity）：**
    *   `∫ [f(x) + g(x)] dx = ∫ f(x) dx + ∫ g(x) dx`
    *   `∫ k * f(x) dx = k * ∫ f(x) dx` （`k` 是常数）
    *   例子：`∫ (3x² + 2cos(x)) dx = 3 * x³/3 + 2 * sin(x) + C = x³ + 2sin(x) + C`

2.  **换元法（Substitution Rule）：**
    这是链式法则在积分中的对应物。如果 `u = g(x)`，那么 `du = g'(x)dx`。
    ```
    ∫ f(g(x)) * g'(x) dx = ∫ f(u) du
    ```
    **例子：** `∫ 2x * e^(x²) dx`
    *   令 `u = x²`，则 `du = 2x dx`。
    *   原积分变为 `∫ e^u du = e^u + C`。
    *   将 `u = x²` 代回，得到 `∫ 2x * e^(x²) dx = e^(x²) + C`。

3.  **分部积分法（Integration by Parts）：**
    这是乘积法则在积分中的对应物。如果 `u = u(x)` 和 `v = v(x)` 都是可微函数，那么 `d(uv) = u dv + v du`。两边积分得到 `uv = ∫ u dv + ∫ v du`，移项得：
    ```
    ∫ u dv = uv - ∫ v du
    ```
    **选择原则（LIATE）：** 通常选择 `u` 为以下列表中靠前的函数类型：
    *   **L**ogarithmic (对数函数)
    *   **I**nverse trigonometric (反三角函数)
    *   **A**lgebraic (代数函数)
    *   **T**rigonometric (三角函数)
    *   **E**xponential (指数函数)
    **例子：** `∫ x * e^x dx`
    *   令 `u = x`，`dv = e^x dx`。
    *   则 `du = dx`，`v = e^x`。
    *   应用公式：`∫ x * e^x dx = x * e^x - ∫ e^x dx = x * e^x - e^x + C = e^x (x - 1) + C`。

## 5. 积分的应用

1.  **计算面积：** 定积分最直接的应用就是计算平面图形的面积。
    *   曲线 `y = f(x)` 与 `x` 轴在 `[a, b]` 之间围成的面积：`A = ∫[a, b] |f(x)| dx`。
    *   两条曲线 `y = f(x)` 和 `y = g(x)` 在 `[a, b]` 之间围成的面积：`A = ∫[a, b] |f(x) - g(x)| dx`。

2.  **计算体积：** 通过旋转曲边梯形可以得到旋转体，其体积可以用定积分计算（圆盘法、壳层法）。

3.  **计算功、能量等物理量：** 物理学中很多量（如变力做功 `W = ∫ F(x) dx`）都可以表示为定积分。

4.  **概率论：** 连续型随机变量的概率密度函数 `f(x)` 在某个区间 `[a, b]` 内取值的概率就是 `P(a ≤ X ≤ b) = ∫[a, b] f(x) dx`。

## 6. 总结

*   **积分**分为定积分和不定积分。定积分计算区间上的累积量（如面积），不定积分寻找原函数。
*   **微积分基本定理**是微分和积分的桥梁：
    *   第一部分：变上限积分的导数是被积函数。
    *   第二部分（牛顿-莱布尼茨公式）：`∫[a, b] f(x) dx = F(b) - F(a)`，其中 `F` 是 `f` 的原函数。
*   掌握基本积分公式和换元法、分部积分法等技巧是计算积分的关键。
*   积分在几何（面积、体积）、物理（功、能量）和概率等领域有广泛应用。
