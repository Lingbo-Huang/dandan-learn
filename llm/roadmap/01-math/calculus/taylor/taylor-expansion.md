# 泰勒展开基础

## 1. 引言

在数学和工程中，我们经常需要处理复杂的函数。有时，直接计算或分析这些函数非常困难。泰勒展开（Taylor Expansion）提供了一种强大的方法，它将一个**光滑**（无穷可微）的函数在某一点附近用一个**多项式**来近似表示。

想象你要用一个简单的多项式来代替一个复杂的函数 `sin(x)` 或 `e^x`，并且希望在 `x=0` 附近这个多项式和原函数非常接近。泰勒展开就是实现这一目标的工具。这个多项式的系数由函数在该点的各阶导数值决定。

## 2. 泰勒公式

### 2.1 泰勒公式的定义

如果函数 `f(x)` 在点 `a` 附近（包含 `a` 的某个开区间内）具有直到 `(n+1)` 阶的导数，那么对于这个区间内的任意 `x`，函数 `f(x)` 可以表示为：

```
f(x) = f(a) + f'(a)(x-a) + f''(a)/2! * (x-a)² + f'''(a)/3! * (x-a)³ + ... + fⁿ(a)/n! * (x-a)ⁿ + Rₙ(x)
```

其中：
*   `f(a), f'(a), f''(a), ..., fⁿ(a)` 分别是函数 `f` 在点 `a` 处的0阶、1阶、2阶、...、n阶导数。
*   `n!` 表示 `n` 的阶乘，即 `n! = n * (n-1) * ... * 2 * 1`。规定 `0! = 1`。
*   `Rₙ(x)` 是**余项**（Remainder），表示用 `n` 次多项式近似 `f(x)` 产生的误差。

这个公式也常写作求和符号的形式：

```
f(x) = Σ (k=0 to n) [fᵏ(a) / k!] * (x-a)ᵏ + Rₙ(x)
```

### 2.2 麦克劳林公式（Maclaurin Series）

当展开点 `a = 0` 时，泰勒公式变为麦克劳林公式，这是最常用的形式之一：

```
f(x) = f(0) + f'(0)x + f''(0)/2! * x² + f'''(0)/3! * x³ + ... + fⁿ(0)/n! * xⁿ + Rₙ(x)
```

或者：

```
f(x) = Σ (k=0 to n) [fᵏ(0) / k!] * xᵏ + Rₙ(x)
```

### 2.3 余项

余项 `Rₙ(x)` 描述了近似的误差。常见的余项形式有两种：

1.  **拉格朗日余项 (Lagrange Form):**
    ```
    Rₙ(x) = f⁽ⁿ⁺¹⁾(ξ) / (n+1)! * (x-a)ⁿ⁺¹
    ```
    其中 `ξ` 是 `a` 和 `x` 之间某个值。

2.  **皮亚诺余项 (Peano Form):**
    ```
    Rₙ(x) = o((x-a)ⁿ)
    ```
    当 `x -> a` 时，`Rₙ(x) / (x-a)ⁿ -> 0`。这表示余项比 `(x-a)ⁿ` 更快地趋于0。

当我们只关心近似多项式本身，而不严格计算误差时，常常会省略余项，写成：

```
f(x) ≈ Σ (k=0 to n) [fᵏ(a) / k!] * (x-a)ᵏ
```

## 3. 常见函数的麦克劳林展开

记住一些基本函数的麦克劳林展开式非常有用。

1.  **指数函数 `e^x`:**
    ```
    e^x = 1 + x + x²/2! + x³/3! + ... + xⁿ/n! + ...
    ```
    收敛域：所有实数 `x`。

2.  **正弦函数 `sin(x)`:**
    ```
    sin(x) = x - x³/3! + x⁵/5! - x⁷/7! + ... + (-1)ᵐ * x^(2m+1) / (2m+1)! + ...
    ```
    （只包含奇数次幂）
    收敛域：所有实数 `x`。

3.  **余弦函数 `cos(x)`:**
    ```
    cos(x) = 1 - x²/2! + x⁴/4! - x⁶/6! + ... + (-1)ᵐ * x^(2m) / (2m)! + ...
    ```
    （只包含偶数次幂）
    收敛域：所有实数 `x`。

4.  **自然对数 `(1+x)`:**
    ```
    ln(1+x) = x - x²/2 + x³/3 - x⁴/4 + ... + (-1)^(n+1) * xⁿ / n + ...
    ```
    收敛域：`-1 < x ≤ 1`。

5.  **几何级数 `(1-x)⁻¹`:**
    ```
    1/(1-x) = 1 + x + x² + x³ + ... + xⁿ + ...
    ```
    收敛域：`|x| < 1`。

## 4. 计算示例

**例子1：** 求 `f(x) = e^x` 在 `a=0` 处的3阶泰勒多项式。

*   `f(0) = e⁰ = 1`
*   `f'(x) = e^x`, `f'(0) = 1`
*   `f''(x) = e^x`, `f''(0) = 1`
*   `f'''(x) = e^x`, `f'''(0) = 1`
*   3阶麦克劳林多项式为：
    ```
    P₃(x) = f(0) + f'(0)x + f''(0)/2! * x² + f'''(0)/3! * x³
          = 1 + x + x²/2 + x³/6
    ```
*   所以 `e^x ≈ 1 + x + x²/2 + x³/6` （当 `x` 接近0时）。

**例子2：** 利用泰勒展开近似计算 `e^(0.1)`。

*   使用 `e^x` 的3阶麦克劳林展开：`e^x ≈ 1 + x + x²/2 + x³/6`
*   令 `x = 0.1`：
    ```
    e^(0.1) ≈ 1 + 0.1 + (0.1)²/2 + (0.1)³/6
            = 1 + 0.1 + 0.01/2 + 0.001/6
            = 1 + 0.1 + 0.005 + 0.000166...
            ≈ 1.105167
    ```
*   真实值 `e^(0.1) ≈ 1.1051709...`，可以看到近似效果很好。

## 5. 泰勒展开的应用

1.  **函数近似计算：** 如上例所示，可以用多项式来近似计算复杂函数的值，这在没有计算器的年代尤其重要，现在在计算机数值计算中也很常见。
2.  **极限计算：** 在计算 `0/0` 或 `∞/∞` 型未定式极限时，泰勒展开可以将分子分母展开成多项式，从而简化计算。
    *   例子：计算 `lim (x->0) [sin(x) - x] / x³`
        *   将 `sin(x)` 展开：`sin(x) ≈ x - x³/6`
        *   代入：`lim (x->0) [x - x³/6 - x] / x³ = lim (x->0) [-x³/6] / x³ = -1/6`
3.  **误差估计：** 通过分析余项 `Rₙ(x)`，可以估计用泰勒多项式近似函数时的误差大小。
4.  **物理和工程中的建模：** 在物理和工程问题中，经常会对复杂的非线性方程进行线性化处理，这本质上就是泰勒展开的一阶近似（忽略高阶小量）。
    *   例如，单摆的运动方程 `d²θ/dt² = -(g/L) * sin(θ)`。当摆角 `θ` 很小时，`sin(θ) ≈ θ`（`sin` 的一阶泰勒展开），方程简化为线性方程 `d²θ/dt² = -(g/L) * θ`，更容易求解。
5.  **优化问题：** 在机器学习和数值优化中，目标函数有时会用其泰勒展开（通常是二阶）来近似，以便找到极值点。例如，牛顿法就是基于函数的二阶泰勒展开。

## 6. 总结

*   泰勒展开将一个光滑函数在某点附近用多项式近似表示。
*   麦克劳林展开是 `a=0` 时的泰勒展开，应用广泛。
*   展开式的系数由函数在该点的各阶导数值决定。
*   余项 `Rₙ(x)` 表示近似的误差。
*   泰勒展开在函数近似、极限计算、误差估计、物理建模和优化等领域有广泛应用。
