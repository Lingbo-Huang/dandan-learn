# 导数与偏导数基础

## 1. 引言

导数（Derivative）和偏导数（Partial Derivative）是微积分学中的核心概念。它们描述了函数在某一点处的变化率，是理解函数行为、优化问题以及机器学习中梯度下降算法等技术的基础。

对于一个Java工程师来说，理解导数就像理解代码中变量的变化率一样。想象一个简单的函数 `y = f(x) = x²`，导数 `dy/dx` 就是 `y` 随着 `x` 变化时的变化速度。在 `x=2` 时，`y=4`，如果 `x` 增加一点点（比如变成2.01），`y` 会增加多少？导数就告诉我们这个“增加多少”的比率。

## 2. 导数（Derivative）

### 2.1 定义

导数描述的是一个**单变量函数** `y = f(x)` 在某一点 `x` 处的瞬时变化率。它衡量的是当自变量 `x` 发生一个微小变化时，因变量 `y` 相应地变化了多少。

**数学定义（极限定义）：**

函数 `f(x)` 在点 `x` 处的导数 `f'(x)` 定义为：

```
f'(x) = lim (h -> 0) [f(x + h) - f(x)] / h
```

如果这个极限存在，我们就说函数 `f(x)` 在点 `x` 处可导。

**通俗理解：**

想象你在开车，`x` 是时间，`f(x)` 是你在时间 `x` 时走过的路程。那么导数 `f'(x)` 就是你的瞬时速度。它不是平均速度（总路程除以总时间），而是某一瞬间的速度。

### 2.2 几何意义

导数的几何意义是函数图像在某一点处**切线的斜率**。

*   **切线：** 通过函数图像上的一点，并且在该点附近与图像“最贴近”的直线。
*   **斜率：** 切线相对于x轴的倾斜程度。斜率为正，函数在该点递增；斜率为负，函数在该点递减；斜率为零，函数在该点可能达到极值（最大值或最小值）。

### 2.3 基本计算规则

为了方便计算，我们有一些基本的求导法则。记住这些法则，可以大大简化求导过程。

1.  **常数法则 (Constant Rule):**
    *   如果 `f(x) = c` （`c` 是常数），那么 `f'(x) = 0`。
    *   解释：常数的值不随 `x` 变化，所以变化率为0。

2.  **幂函数法则 (Power Rule):**
    *   如果 `f(x) = x^n` （`n` 是任意实数），那么 `f'(x) = n * x^(n-1)`。
    *   例子：
        *   `f(x) = x^3`，则 `f'(x) = 3 * x^2`。
        *   `f(x) = x`，则 `f'(x) = 1 * x^0 = 1`。
        *   `f(x) = 1/x = x^(-1)`，则 `f'(x) = -1 * x^(-2) = -1/x^2`。
        *   `f(x) = √x = x^(1/2)`，则 `f'(x) = (1/2) * x^(-1/2) = 1/(2√x)`。

3.  **常数倍数法则 (Constant Multiple Rule):**
    *   如果 `f(x) = c * g(x)` （`c` 是常数），那么 `f'(x) = c * g'(x)`。
    *   解释：常数因子可以提到导数符号外面。

4.  **和差法则 (Sum/Difference Rule):**
    *   如果 `f(x) = g(x) ± h(x)`，那么 `f'(x) = g'(x) ± h'(x)`。
    *   解释：两个函数的和（或差）的导数等于各自导数的和（或差）。

5.  **乘积法则 (Product Rule):**
    *   如果 `f(x) = g(x) * h(x)`，那么 `f'(x) = g'(x) * h(x) + g(x) * h'(x)`。
    *   例子：`f(x) = x^2 * sin(x)`。
        *   `g(x) = x^2`，`g'(x) = 2x`。
        *   `h(x) = sin(x)`，`h'(x) = cos(x)`。
        *   `f'(x) = 2x * sin(x) + x^2 * cos(x)`。

6.  **商法则 (Quotient Rule):**
    *   如果 `f(x) = g(x) / h(x)`，那么 `f'(x) = [g'(x) * h(x) - g(x) * h'(x)] / [h(x)]^2`。
    *   例子：`f(x) = (x^2 + 1) / x`。
        *   `g(x) = x^2 + 1`，`g'(x) = 2x`。
        *   `h(x) = x`，`h'(x) = 1`。
        *   `f'(x) = [2x * x - (x^2 + 1) * 1] / x^2 = [2x^2 - x^2 - 1] / x^2 = (x^2 - 1) / x^2`。

7.  **链式法则 (Chain Rule):**
    *   如果 `y = f(u)` 且 `u = g(x)`，那么 `dy/dx = (dy/du) * (du/dx)`。
    *   这个法则非常重要，尤其在处理复合函数时。我们将在后续章节详细讨论。

### 2.4 常见函数的导数

记住一些基本函数的导数，是快速求导的基础。

*   `d/dx (x^n) = n * x^(n-1)` （幂函数法则）
*   `d/dx (e^x) = e^x`
*   `d/dx (ln(x)) = 1/x` （`x > 0`）
*   `d/dx (sin(x)) = cos(x)`
*   `d/dx (cos(x)) = -sin(x)`
*   `d/dx (tan(x)) = sec^2(x) = 1 + tan^2(x)`

### 2.5 高阶导数

导数 `f'(x)` 本身也是一个函数。我们可以对 `f'(x)` 再次求导，得到 `f''(x)`，称为**二阶导数**。同理，可以得到三阶导数 `f'''(x)`，四阶导数 `f''''(x)`，等等。

*   **一阶导数 `f'(x)`：** 描述函数 `f(x)` 的变化率（斜率）。
*   **二阶导数 `f''(x)`：** 描述一阶导数 `f'(x)` 的变化率，即函数 `f(x)` 的**曲率**或**加速度**。
    *   如果 `f''(x) > 0`，则函数图像在该点是“凹向上”的（像碗）。
    *   如果 `f''(x) < 0`，则函数图像在该点是“凹向下”的（像山峰）。
*   **记法：** `f''(x)`, `d²y/dx²`, `y''`。

例子：`f(x) = x^3`
*   一阶导数：`f'(x) = 3x^2`
*   二阶导数：`f''(x) = d/dx (3x^2) = 6x`
*   三阶导数：`f'''(x) = d/dx (6x) = 6`

## 3. 偏导数（Partial Derivative）

### 3.1 引入

在现实世界中，很多现象依赖于**多个变量**。例如，一个房间的温度 `T` 可能同时依赖于位置 `x`、`y`、`z` 和时间 `t`，即 `T = f(x, y, z, t)`。我们如何描述温度在某个方向上的变化率呢？这就需要引入偏导数。

### 3.2 定义

偏导数描述的是一个**多变量函数** `z = f(x, y, ...)` 在其他变量保持不变的情况下，关于其中一个变量的变化率。

**数学定义：**

对于二元函数 `z = f(x, y)`，其关于 `x` 的偏导数定义为：

```
∂f/∂x = lim (h -> 0) [f(x + h, y) - f(x, y)] / h
```

其关于 `y` 的偏导数定义为：

```
∂f/∂y = lim (h -> 0) [f(x, y + h) - f(x, y)] / h
```

符号 `∂` 读作 "partial"，用来区分偏导数和普通导数。

**通俗理解：**

计算 `∂f/∂x` 时，我们把 `y` 看作一个常数，只考虑 `x` 的微小变化对 `f` 的影响。就像在三维地形图上，你站在一个点，只沿着东西方向（x轴）走一小步，看看高度（z值）变化了多少，这个变化率就是 `∂f/∂x`。

### 3.3 计算

计算偏导数的方法与计算普通导数类似，只是在对一个变量求导时，把其他所有变量都当作常数。

**例子：** `f(x, y) = x^2 * y + sin(x * y)`

*   求 `∂f/∂x`：把 `y` 看作常数。
    *   `∂/∂x (x^2 * y) = y * ∂/∂x (x^2) = y * 2x = 2xy` （常数倍数法则和幂函数法则）
    *   `∂/∂x (sin(x * y)) = cos(x * y) * ∂/∂x (x * y) = cos(x * y) * y` （链式法则，把 `x * y` 看作 `u`）
    *   所以，`∂f/∂x = 2xy + y * cos(xy)`

*   求 `∂f/∂y`：把 `x` 看作常数。
    *   `∂/∂y (x^2 * y) = x^2 * ∂/∂y (y) = x^2 * 1 = x^2` （常数倍数法则和幂函数法则）
    *   `∂/∂y (sin(x * y)) = cos(x * y) * ∂/∂y (x * y) = cos(x * y) * x` （链式法则，把 `x * y` 看作 `u`）
    *   所以，`∂f/∂y = x^2 + x * cos(xy)`

### 3.4 高阶偏导数

和普通导数一样，偏导数也可以继续求导。

对于 `z = f(x, y)`，我们有：
*   一阶偏导数：`∂f/∂x`, `∂f/∂y`
*   二阶偏导数：
    *   `∂²f/∂x²`：对 `x` 求两次偏导。
    *   `∂²f/∂y²`：对 `y` 求两次偏导。
    *   `∂²f/∂x∂y`：先对 `y` 求偏导，再对 `x` 求偏导。
    *   `∂²f/∂y∂x`：先对 `x` 求偏导，再对 `y` 求偏导。

**混合偏导数相等定理（Clairaut's Theorem）：**
如果函数 `f` 的二阶混合偏导数 `∂²f/∂x∂y` 和 `∂²f/∂y∂x` 在某区域内连续，那么在这区域内这两个混合偏导数相等。

**例子：** `f(x, y) = x^3 * y^2`
*   `∂f/∂x = 3x^2 * y^2`
*   `∂f/∂y = x^3 * 2y = 2x^3 * y`
*   `∂²f/∂x² = ∂/∂x (3x^2 * y^2) = 6x * y^2`
*   `∂²f/∂y² = ∂/∂y (2x^3 * y) = 2x^3`
*   `∂²f/∂x∂y = ∂/∂x (2x^3 * y) = 6x^2 * y`
*   `∂²f/∂y∂x = ∂/∂y (3x^2 * y^2) = 6x^2 * y`
*   可以看到 `∂²f/∂x∂y = ∂²f/∂y∂x`。

## 4. 总结

*   **导数** `f'(x)` 是单变量函数 `f(x)` 的变化率，几何上是切线斜率。
*   **偏导数** `∂f/∂x` 是多变量函数 `f(x, y, ...)` 在其他变量不变时，对某个变量的变化率。
*   掌握基本的求导法则（幂函数、常数、和差、乘积、商、链式法则）是计算导数和偏导数的关键。
*   **高阶导数/偏导数**描述了变化率的变化率，对于理解函数的曲率等性质很重要。
*   这些概念是后续学习梯度、泰勒展开以及机器学习中优化算法（如梯度下降）的基石。
