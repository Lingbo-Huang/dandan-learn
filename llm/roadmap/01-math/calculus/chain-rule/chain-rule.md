# 链式法则基础

## 1. 引言

在前面学习导数和偏导数的计算规则时，我们接触了一些基本函数（如 `x^n`, `sin(x)`, `e^x` 等）的导数以及它们的组合规则（和、差、积、商）。然而，在实际问题和机器学习模型中，我们遇到的函数往往是**复合函数**（Composite Function），即一个函数的输入是另一个函数的输出。

例如，函数 `y = sin(x²)` 可以看作是 `y = sin(u)` 和 `u = x²` 的复合。如何求这样的复合函数的导数呢？这就是链式法则（Chain Rule）要解决的问题。链式法则是微积分中最重要的求导法则之一，对于理解和计算复杂函数的导数至关重要，尤其是在反向传播算法中。

## 2. 链式法则（单变量）

### 2.1 定义与公式

如果 `y = f(u)` 且 `u = g(x)`，那么 `y` 通过 `u` 间接地依赖于 `x`，形成复合函数 `y = f(g(x))`。链式法则告诉我们如何求这个复合函数对 `x` 的导数。

**链式法则公式：**

```
dy/dx = (dy/du) * (du/dx)
```

或者写作：

```
d/dx [f(g(x))] = f'(g(x)) * g'(x)
```

**通俗理解：**

想象你在开车 (`x` 是时间)，车速表显示的是 `dx/dt`（位置对时间的导数）。现在你看到一个温度计 (`y` 是温度)，它显示的温度取决于你的海拔高度 (`u` 是海拔)。海拔高度又取决于你的位置 (`u = g(x)`)。那么，温度随时间的变化率 `dy/dt` 就等于温度对海拔的变化率 `dy/du` 乘以海拔对位置的变化率 `du/dx` 再乘以位置对时间的变化率 `dx/dt`。链式法则就是把这种“链式”的变化率相乘起来。

### 2.2 计算示例

**例子1：** 求 `y = (3x + 2)^4` 的导数。

*   识别内层函数和外层函数：
    *   内层：`u = g(x) = 3x + 2`
    *   外层：`y = f(u) = u^4`
*   分别求导：
    *   `du/dx = g'(x) = 3`
    *   `dy/du = f'(u) = 4u^3`
*   应用链式法则：
    *   `dy/dx = (dy/du) * (du/dx) = 4u^3 * 3 = 12u^3`
*   将 `u = 3x + 2` 代入回去：
    *   `dy/dx = 12(3x + 2)^3`

**例子2：** 求 `y = e^(sin(x))` 的导数。

*   识别内层函数和外层函数：
    *   内层：`u = g(x) = sin(x)`
    *   外层：`y = f(u) = e^u`
*   分别求导：
    *   `du/dx = g'(x) = cos(x)`
    *   `dy/du = f'(u) = e^u`
*   应用链式法则：
    *   `dy/dx = (dy/du) * (du/dx) = e^u * cos(x)`
*   将 `u = sin(x)` 代入回去：
    *   `dy/dx = e^(sin(x)) * cos(x)`

## 3. 链式法则（多变量）

在多变量微积分中，函数关系会更加复杂。一个变量可能通过多条路径影响最终结果。链式法则需要相应地扩展。

### 3.1 全导数（Total Derivative）

如果 `z = f(x, y)`，而 `x = g(t)` 和 `y = h(t)` 都是 `t` 的函数，那么 `z` 最终也是 `t` 的函数。`z` 对 `t` 的导数（全导数）为：

```
dz/dt = (∂f/∂x) * (dx/dt) + (∂f/∂y) * (dy/dt)
```

**解释：** `z` 的变化通过两条路径传递：一条是通过 `x` 的变化，另一条是通过 `y` 的变化。总的变化率是这两条路径贡献的总和。

**例子：** `z = x² * y`, `x = t³`, `y = t²`。求 `dz/dt`。

*   `∂f/∂x = 2xy`, `dx/dt = 3t²`
*   `∂f/∂y = x²`, `dy/dt = 2t`
*   `dz/dt = (2xy) * (3t²) + (x²) * (2t) = 2 * t³ * t² * 3t² + (t³)² * 2t = 6t^7 + 2t^7 = 8t^7`

### 3.2 偏导数形式的链式法则

如果 `z = f(x, y)`，而 `x = g(s, t)` 和 `y = h(s, t)` 都是 `s` 和 `t` 的函数，那么 `z` 也是 `s` 和 `t` 的函数。我们可以求 `z` 对 `s` 和 `t` 的偏导数。

*   `z` 对 `s` 的偏导数：
    ```
    ∂z/∂s = (∂f/∂x) * (∂x/∂s) + (∂f/∂y) * (∂y/∂s)
    ```
*   `z` 对 `t` 的偏导数：
    ```
    ∂z/∂t = (∂f/∂x) * (∂x/∂t) + (∂f/∂y) * (∂y/∂t)
    ```

**例子：** `z = x² * y`, `x = s + t`, `y = s - t`。求 `∂z/∂s` 和 `∂z/∂t`。

*   `∂f/∂x = 2xy`, `∂x/∂s = 1`, `∂x/∂t = 1`
*   `∂f/∂y = x²`, `∂y/∂s = 1`, `∂y/∂t = -1`
*   `∂z/∂s = (2xy) * (1) + (x²) * (1) = 2xy + x² = 2(s+t)(s-t) + (s+t)² = 2(s² - t²) + (s² + 2st + t²) = 2s² - 2t² + s² + 2st + t² = 3s² + 2st - t²`
*   `∂z/∂t = (2xy) * (1) + (x²) * (-1) = 2xy - x² = 2(s+t)(s-t) - (s+t)² = 2(s² - t²) - (s² + 2st + t²) = 2s² - 2t² - s² - 2st - t² = s² - 2st - 3t²`

## 4. 链式法则在机器学习中的应用（反向传播）

链式法则是深度学习中**反向传播算法**（Backpropagation）的数学基础。神经网络可以看作是一个由许多简单函数（如线性变换、激活函数）复合而成的复杂函数。计算损失函数相对于网络中每个参数的梯度时，就需要反复应用链式法则，将误差从输出层逐层向输入层传播，并计算每一层参数的梯度。

例如，对于一个简单的两层网络：
`L` (损失) -> `z2` (第二层加权和) -> `a2` (第二层激活) -> `z1` (第一层加权和) -> `a1` (第一层激活) -> `x` (输入)

计算 `L` 对第一层权重 `W1` 的梯度时，会用到：
`∂L/∂W1 = (∂L/∂a2) * (∂a2/∂z2) * (∂z2/∂a1) * (∂a1/∂z1) * (∂z1/∂W1)`

这就是链式法则的连续应用。

## 5. 总结

*   **单变量链式法则：** `dy/dx = (dy/du) * (du/dx)`，用于求解形如 `y = f(g(x))` 的复合函数的导数。
*   **多变量链式法则：** 用于处理一个变量通过多条路径影响最终结果的情况。
    *   全导数：`dz/dt = (∂f/∂x) * (dx/dt) + (∂f/∂y) * (dy/dt)`
    *   偏导数形式：`∂z/∂s = (∂f/∂x) * (∂x/∂s) + (∂f/∂y) * (∂y/∂s)`
*   链式法则是计算复杂函数导数的强大工具，是理解机器学习中梯度计算（尤其是反向传播）的关键。
