# 向量空间 (Vector Space) 详解

向量空间（也称为线性空间）是线性代数中的一个核心概念。它提供了一个抽象的框架来研究向量及其运算的普遍性质，而不仅仅局限于我们熟悉的几何向量或坐标向量。

## 1. 向量空间的定义

一个**向量空间 (Vector Space)** V 是一个集合，其中的元素称为**向量**，并且在这个集合上定义了两种运算：
1.  **向量加法 (Vector Addition)**: 任意两个向量 **u** 和 **v** 可以相加，得到另一个向量 **u + v**，这个结果也在 V 中。
2.  **标量乘法 (Scalar Multiplication)**: 任意一个向量 **u** 可以与一个标量（通常指实数或复数）**c** 相乘，得到另一个向量 **c * u**，这个结果也在 V 中。

为了成为一个向量空间，这两种运算必须满足以下**八条公理**（对于所有向量 **u, v, w** 属于 V 和所有标量 **c, d**）：

**向量加法的公理：**
1.  **封闭性 (Closure under Addition)**: **u + v** 属于 V。
2.  **结合律 (Associativity)**: (**u + v**) + **w** = **u** + (**v** + **w**)
3.  **交换律 (Commutativity)**: **u + v** = **v + u**
4.  **零向量 (Zero Vector)**: 存在一个唯一的向量 **0** 属于 V，使得对于 V 中的任意向量 **u**，都有 **u + 0** = **u**。
5.  **负向量 (Additive Inverse)**: 对于 V 中的每一个向量 **u**，都存在一个向量 **-u** 属于 V，使得 **u + (-u)** = **0**。

**标量乘法的公理：**
6.  **封闭性 (Closure under Scalar Multiplication)**: **c * u** 属于 V。
7.  **标量乘法与向量加法的分配律 (Distributivity of Scalar Multiplication with Respect to Vector Addition)**: **c * (u + v)** = **c * u + c * v**
8.  **标量乘法与标量加法的分配律 (Distributivity of Scalar Multiplication with Respect to Scalar Addition)**: **(c + d) * u** = **c * u + d * u**
9.  **标量乘法与标量乘法的结合律 (Associativity of Scalar Multiplication)**: **c * (d * u)** = **(c * d) * u**
10. **标量乘法的单位元 (Scalar Multiplicative Identity)**: **1 * u** = **u**，其中 1 是标量域中的乘法单位元。

**常见例子：**
*   **Rⁿ**: 所有 n 维实向量的集合，是我们最熟悉的向量空间。
*   **Cⁿ**: 所有 n 维复向量的集合。
*   **Pₙ**: 所有次数小于或等于 n 的实系数多项式的集合。
*   **M_{m×n}**: 所有 m×n 实矩阵的集合。

## 2. 子空间 (Subspace)

如果向量空间 V 的一个子集 H 本身对于 V 中定义的加法和标量乘法也构成一个向量空间，那么 H 就称为 V 的一个**子空间 (Subspace)**。

**判定定理：**
要判断一个集合 H 是否是向量空间 V 的子空间，只需要验证以下三个条件：
1.  **包含零向量**: V 中的零向量 **0** 也在 H 中。
2.  **对加法封闭**: 如果 **u** 和 **v** 都在 H 中，那么 **u + v** 也在 H 中。
3.  **对标量乘法封闭**: 如果 **u** 在 H 中，**c** 是任意标量，那么 **c * u** 也在 H 中。

**重要例子（与矩阵相关的子空间）：**
对于一个 m×n 矩阵 A，可以定义四个重要的子空间：

### 2.1 列空间 (Column Space)

矩阵 A 的**列空间 (Column Space)**，记作 **Col A**，是由 A 的所有列向量的所有线性组合构成的集合。它是 **R^m** 的一个子空间。

**例子：**
如果 A = [a₁ a₂ ... aₙ]，其中 aᵢ 是 A 的第 i 列，那么 Col A = Span{a₁, a₂, ..., aₙ}。

### 2.2 零空间 (Null Space) 或 核 (Kernel)

矩阵 A 的**零空间 (Null Space)**，记作 **Nul A**，是齐次线性方程组 **Ax = 0** 的所有解 **x** 构成的集合。它是 **R^n** 的一个子空间。

**例子：**
如果 A = [1 2; 3 6]，那么方程 Ax = 0 就是：
```
x₁ + 2x₂ = 0
3x₁ + 6x₂ = 0
```
这两个方程实际上是等价的。解为 x₁ = -2x₂。所以解集（零空间）是所有形如 [-2x₂; x₂] = x₂[-2; 1] 的向量，即由向量 [-2; 1] 张成的一维子空间。

### 2.3 行空间 (Row Space)

矩阵 A 的**行空间 (Row Space)**，记作 **Row A**，是由 A 的所有行向量的所有线性组合构成的集合。由于矩阵的行向量是 **R^n** 中的向量，所以行空间是 **R^n** 的一个子空间。

**重要性质：**
行空间等于 A 的转置矩阵 A^T 的列空间，即 **Row A = Col (A^T)**。

### 2.4 左零空间 (Left Null Space)

矩阵 A 的**左零空间 (Left Null Space)**，记作 **Nul (A^T)**，是齐次线性方程组 **A^T y = 0** 的所有解 **y** 构成的集合。它是 **R^m** 的一个子空间。

名称“左零空间”来源于：方程 **A^T y = 0** 可以写成 **y^T A = 0**，即向量 **y^T**（**y** 的转置）从左边“乘”矩阵 A 得到零向量。

## 3. 基 (Basis) 与 维数 (Dimension)

### 3.1 基 (Basis)

向量空间 V 的一个**基 (Basis)** 是 V 中的一个向量组，它满足两个条件：
1.  **线性无关 (Linearly Independent)**。
2.  **张成 V (Span V)**：V 中的任意向量都可以表示为这个向量组的线性组合。

**重要性质：**
*   基不是唯一的，但一个向量空间的所有基都包含相同数目的向量。
*   如果一个向量空间 V 有一个基包含 n 个向量，那么 V 中任意 n+1 个向量都线性相关。

### 3.2 维数 (Dimension)

向量空间 V 的**维数 (Dimension)**，记作 **dim V**，是其任意一个基中向量的个数。

**例子：**
*   **Rⁿ** 的维数是 n。标准基 {e₁, e₂, ..., eₙ} 是一个包含 n 个向量的基。
*   零空间 {0} 的维数定义为 0。
*   对于多项式空间 Pₙ（次数小于或等于 n 的多项式），其维数是 n+1。一个基是 {1, x, x², ..., xⁿ}。

### 3.3 矩阵的秩 (Rank) 与子空间维数的关系

对于一个 m×n 矩阵 A，其**秩 (rank)** 定义为 A 的列空间（或行空间）的维数，记作 **rank A**。

矩阵 A 及其四个基本子空间的维数之间存在以下重要关系（秩-零化度定理）：

*   **rank A + dim(Nul A) = n** （列数）
*   **rank A + dim(Nul A^T) = m** （行数）

**解释：**
*   **dim(Col A) = rank A**
*   **dim(Nul A) = n - rank A**
*   **dim(Row A) = rank A**
*   **dim(Nul A^T) = m - rank A**

这些关系深刻地揭示了矩阵的结构和性质。

## 4. 坐标系 (Coordinate System)

一旦在向量空间 V 中选定一个基 **B = {b₁, b₂, ..., bₚ}**，那么 V 中的任意向量 **x** 都可以唯一地表示为这个基的线性组合：
```
x = c₁b₁ + c₂b₂ + ... + cₚbₚ
```
这组系数 **[x]_B = [c₁; c₂; ...; cₚ]** 称为向量 **x** 在基 **B** 下的**坐标向量 (Coordinate Vector)**。

**例子：**
在 R² 中，标准基是 E = {e₁=(1,0), e₂=(0,1)}。向量 x = (3, 4) 在标准基下的坐标向量就是 [x]_E = [3; 4]。

如果我们选择另一个基 B = {b₁=(2,1), b₂=(1,1)}，那么同一个向量 x = (3, 4) 在基 B 下的坐标向量 [x]_B = [c₁; c₂] 需要满足：
```
c₁*(2,1) + c₂*(1,1) = (3,4)
```
解这个方程组可得 c₁ = -1, c₂ = 5。所以 [x]_B = [-1; 5]。

**重要性：**
通过坐标系，我们可以将抽象的向量空间中的问题转化为我们熟悉的 Rⁿ 中的具体坐标运算，这为理论分析和实际计算提供了极大的便利。
