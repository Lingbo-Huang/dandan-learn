# 线性变换 (Linear Transformation) 详解

线性变换是线性代数中连接向量空间和矩阵的核心概念。它描述了向量空间中向量的一种特定类型的映射规则。

## 1. 线性变换的定义

设 V 和 W 是两个向量空间。一个从 V 到 W 的**变换 (Transformation)** T 是一个规则，它将 V 中的每一个向量 **x** 都对应到 W 中唯一的一个向量 **T(x)**。V 称为**定义域 (Domain)**，W 称为**余定义域 (Codomain)**。**T(x)** 称为 **x** 的**像 (Image)**，所有像 **T(x)** 构成的集合称为 T 的**值域 (Range)** 或**像空间 (Image Space)**。

一个变换 **T** 被称为**线性变换 (Linear Transformation)**，当且仅当对于 V 中的任意两个向量 **u** 和 **v**，以及任意标量 c，它都满足以下两个性质：

1.  **可加性 (Additivity)**: **T(u + v) = T(u) + T(v)**
2.  **齐次性 (Homogeneity)**: **T(c * u) = c * T(u)**

**几何直观：**
线性变换具有以下几何特征：
*   变换前是直线的，变换后依然是直线。
*   原点保持不变，即 **T(0) = 0**。
*   网格线保持平行且等距分布。

**例子：**
*   **恒等变换**: **T(x) = x**。这是一个线性变换。
*   **零变换**: **T(x) = 0**。这是一个线性变换。
*   **旋转变换**: 将平面内的向量绕原点逆时针旋转 θ 角度。这是一个线性变换。
*   **投影变换**: 将三维空间中的向量投影到某个平面上。这是一个线性变换。
*   **函数的导数**: 将可微函数映射为其导数的变换 D(f) = f' 是一个线性变换（定义域和余定义域都是函数空间）。

## 2. 线性变换的矩阵表示

线性变换与矩阵之间存在着深刻的联系。一旦我们为向量空间选择了基，线性变换就可以用矩阵来表示。

### 2.1 从几何到代数：用矩阵描述线性变换

考虑从 **Rⁿ** 到 **R^m** 的线性变换 T。为了用矩阵表示 T，我们首先需要选择 **Rⁿ** 的一个基 **B = {b₁, b₂, ..., bₙ}** 和 **R^m** 的一个基 **C = {c₁, c₂, ..., cₘ}**。

对于 **Rⁿ** 的标准基 **E = {e₁, e₂, ..., eₙ}**，其中 **eⱼ** 是第 j 个分量为1，其余分量为0的向量。

我们将基向量 **e₁, e₂, ..., eₙ** 经过变换 T 后的像 **T(e₁), T(e₂), ..., T(eₙ)** 分别表示为 **R^m** 中的向量。

由于 **T(eⱼ)** 是 **R^m** 中的向量，它可以写成 **R^m** 标准基 **{f₁, f₂, ..., fₘ}** 的线性组合：

```
T(e₁) = a₁₁f₁ + a₂₁f₂ + ... + aₘ₁fₘ
T(e₂) = a₁₂f₁ + a₂₂f₂ + ... + aₘ₂fₘ
...
T(eₙ) = a₁ₙf₁ + a₂ₙf₂ + ... + aₘₙfₘ
```

我们将这些像向量 **T(e₁), T(e₂), ..., T(eₙ)** 的坐标（作为列向量）并排放在一起，构成一个 m×n 矩阵 A：

```
A = [T(e₁) T(e₂) ... T(eₙ)]
  = [a₁₁ a₁₂ ... a₁ₙ]
    [a₂₁ a₂₂ ... a₂ₙ]
    [ .   .  ...  . ]
    [aₘ₁ aₘ₂ ... aₘₙ]
```

这个矩阵 A 就称为线性变换 T **相对于标准基的矩阵表示**。

**重要性质：**
对于 **Rⁿ** 中的任意向量 **x**，线性变换 T(x) 的结果等于矩阵 A 与向量 x 的乘积：
```
T(x) = A * x
```

**例子：**
考虑 **R²** 中的旋转变换 T，将向量逆时针旋转 90 度。
*   标准基向量 e₁ = (1, 0) 变换后成为 T(e₁) = (0, 1)。
*   标准基向量 e₂ = (0, 1) 变换后成为 T(e₂) = (-1, 0)。
*   因此，变换 T 相对于标准基的矩阵表示为：
    ```
    A = [T(e₁) T(e₂)] = [0 -1]
                        [1  0]
    ```
*   对于任意向量 x = (x₁, x₂)，其变换结果为：
    ```
    T(x) = A * x = [0 -1] [x₁] = [-x₂]
                   [1  0] [x₂]   [ x₁]
    ```

### 2.2 一般情况下的矩阵表示

如果选择了 **Rⁿ** 的非标准基 **B = {b₁, b₂, ..., bₙ}** 和 **R^m** 的基 **C = {c₁, c₂, ..., cₘ}**，那么线性变换 T 的矩阵表示会有所不同。

我们计算 T(b₁), T(b₂), ..., T(bₙ)，并将它们分别用基 C 表示：
```
[T(b₁)]_C = [a₁₁; a₂₁; ...; aₘ₁]
[T(b₂)]_C = [a₁₂; a₂₂; ...; aₘ₂]
...
[T(bₙ)]_C = [a₁ₙ; a₂ₙ; ...; aₘₙ]
```

然后构造矩阵：
```
[T]_B^C = [[T(b₁)]_C [T(b₂)]_C ... [T(bₙ)]_C]
        = [a₁₁ a₁₂ ... a₁ₙ]
          [a₂₁ a₂₂ ... a₂ₙ]
          [ .   .  ...  . ]
          [aₘ₁ aₘ₂ ... aₘₙ]
```
这个矩阵 **[T]_B^C** 称为线性变换 T **相对于基 B（定义域）和基 C（余定义域）的矩阵表示**。

对于 V 中的任意向量 **x**，有：
```
[T(x)]_C = [T]_B^C * [x]_B
```

## 3. 线性变换的核 (Kernel) 与像 (Image)

线性变换 T: V -> W 的两个重要子空间是它的核和像。

### 3.1 核 (Kernel) 或 零空间 (Null Space)

线性变换 T 的**核 (Kernel)**，记作 **Ker(T)** 或 **Nul(T)**，是所有被 T 映射到零向量的向量 **x** 组成的集合：
```
Ker(T) = {x ∈ V | T(x) = 0_W}
```
其中 0_W 是 W 中的零向量。

**性质：**
*   Ker(T) 是 V 的一个子空间。
*   如果 T 由矩阵 A 表示（T(x) = Ax），那么 Ker(T) 就是矩阵 A 的零空间 Nul(A)。

### 3.2 像 (Image) 或 列空间 (Column Space)

线性变换 T 的**像 (Image)**，记作 **Im(T)** 或 **Range(T)**，是所有像 **T(x)** （其中 x ∈ V）组成的集合：
```
Im(T) = {T(x) | x ∈ V}
```

**性质：**
*   Im(T) 是 W 的一个子空间。
*   如果 T 由矩阵 A 表示（T(x) = Ax），那么 Im(T) 就是矩阵 A 的列空间 Col(A)。

### 3.3 维数定理 (Rank-Nullity Theorem)

对于从 n 维向量空间 V 到 m 维向量空间 W 的线性变换 T，有以下重要关系：
```
dim(Ker(T)) + dim(Im(T)) = dim(V) = n
```
如果 T 由 m×n 矩阵 A 表示，则：
```
dim(Nul(A)) + dim(Col(A)) = n
```
这与之前在矩阵部分提到的秩-零化度定理是一致的。

## 4. 坐标变换 (Change of Basis)

当我们从一个基变换到另一个基时，向量的坐标也会相应地改变。这种坐标之间的转换也是一种线性变换。

设 V 是一个 n 维向量空间，B = {b₁, b₂, ..., bₙ} 和 C = {c₁, c₂, ..., cₙ} 是 V 的两组基。

对于 V 中的任意向量 **x**，它在基 B 下的坐标是 **[x]_B**，在基 C 下的坐标是 **[x]_C**。

存在一个唯一的 n×n 可逆矩阵 **P_B->C**，使得：
```
[x]_C = P_B->C * [x]_B
```
这个矩阵 **P_B->C** 称为**从基 B 到基 C 的坐标变换矩阵 (Change of Basis Matrix)**。

**构造方法：**
坐标变换矩阵 **P_B->C** 的列向量是基 B 中的向量 **b₁, b₂, ..., bₙ** 在基 C 下的坐标：
```
P_B->C = [[b₁]_C [b₂]_C ... [bₙ]_C]
```

**例子：**
在 R² 中，标准基是 E = {e₁=(1,0), e₂=(0,1)}，另一组基是 B = {b₁=(2,1), b₂=(1,1)}。

向量 x = (3, 4) 在标准基下的坐标是 [x]_E = [3; 4]。

要找到从基 B 到标准基 E 的变换矩阵 P_B->E，我们需要找到 b₁ 和 b₂ 在标准基 E 下的坐标，这其实就是 b₁ 和 b₂ 本身：
```
P_B->E = [[b₁]_E [b₂]_E] = [2 1]
                            [1 1]
```

所以，x 在基 B 下的坐标 [x]_B 满足：
```
[x]_E = P_B->E * [x]_B
[3]     [2 1]   [c₁]
[4]  =  [1 1] * [c₂]
```
解这个方程组得到 [x]_B = [c₁; c₂] = [-1; 5]。

反之，从标准基 E 到基 B 的变换矩阵 P_E->B 是 P_B->E 的逆矩阵。

**重要性：**
坐标变换允许我们在不同的基之间灵活转换，这对于理解同一个线性变换在不同视角下的表现、简化矩阵形式（如对角化）等都非常有用。

## 5. 线性变换在机器学习中的应用

线性变换在机器学习中无处不在：
*   **数据预处理**：标准化、白化等操作是线性变换。
*   **降维**：主成分分析(PCA)的核心是找到一个线性变换，将数据投影到方差最大的方向上。
*   **神经网络**：每一层（不包括激活函数）本质上都是一个线性变换（矩阵乘法）。
*   **图像处理**：旋转、缩放、裁剪等几何变换都是线性变换。
*   **特征工程**：线性变换可以用来构造新的特征。

理解线性变换及其矩阵表示是掌握更高级机器学习算法和深度学习架构的关键。
