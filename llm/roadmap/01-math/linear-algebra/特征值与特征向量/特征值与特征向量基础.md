# 特征值 (Eigenvalue) 与 特征向量 (Eigenvector) 详解

特征值和特征向量是线性代数中极其重要且应用广泛的概念。它们揭示了矩阵（或线性变换）的本质特性，使我们能够深入理解矩阵的行为，并简化许多复杂的计算。

## 1. 特征值与特征向量的定义

对于一个 n×n 的方阵 A，如果存在一个标量 λ（读作 lambda）和一个非零向量 **v**，使得：
```
A * v = λ * v
```
那么：
*   标量 λ 称为矩阵 A 的一个**特征值 (Eigenvalue)**。
*   非零向量 **v** 称为矩阵 A 对应于（或属于）特征值 λ 的一个**特征向量 (Eigenvector)**。

**几何直观：**
特征向量是在线性变换 A 作用下，方向保持不变（或反向，即方向改变180度）的向量。特征值 λ 则表示特征向量 **v** 在变换后被拉伸或压缩的比例。
*   如果 λ > 1，特征向量被拉长。
*   如果 0 < λ < 1，特征向量被缩短。
*   如果 λ < 0，特征向量不仅被缩放，方向还发生了反转。
*   如果 λ = 0，特征向量被变换为零向量。

**例子：**
考虑矩阵 A = [2 0; 0 3]。
*   向量 v₁ = [1; 0] 经过 A 变换后得到 A*v₁ = [2; 0] = 2*[1; 0]。所以 λ₁ = 2 是 A 的一个特征值，v₁ = [1; 0] 是对应的一个特征向量。
*   向量 v₂ = [0; 1] 经过 A 变换后得到 A*v₂ = [0; 3] = 3*[0; 1]。所以 λ₂ = 3 是 A 的一个特征值，v₂ = [0; 1] 是对应的一个特征向量。

## 2. 特征方程与特征多项式

为了系统地求解特征值和特征向量，我们需要引入特征方程和特征多项式。

将定义式 A * v = λ * v 移项并整理：
```
A * v - λ * v = 0
(A - λ * I) * v = 0
```
其中 I 是 n×n 的单位矩阵。

这是一个齐次线性方程组。我们要求的是**非零**解 **v**。根据线性代数的理论，齐次线性方程组存在非零解的充要条件是其系数矩阵的行列式为零。因此，λ 是 A 的特征值的充要条件是：
```
det(A - λ * I) = 0
```

这个关于 λ 的方程称为矩阵 A 的**特征方程 (Characteristic Equation)**。
行列式 det(A - λ * I) 是一个关于 λ 的 n 次多项式，称为矩阵 A 的**特征多项式 (Characteristic Polynomial)**。

**例子：**
继续上面 A = [2 0; 0 3] 的例子。
*   计算 A - λ * I：
    ```
    A - λ * I = [2 0; 0 3] - [λ 0; 0 λ] = [2-λ   0]
                                          [  0  3-λ]
    ```
*   计算特征多项式：
    ```
    det(A - λ * I) = (2-λ)(3-λ) - 0*0 = (2-λ)(3-λ)
    ```
*   解特征方程 det(A - λ * I) = 0：
    ```
    (2-λ)(3-λ) = 0
    ```
    解得 λ₁ = 2, λ₂ = 3。这与我们之前观察到的结果一致。

## 3. 求解特征向量

一旦求得特征值 λᵢ，就可以通过求解齐次线性方程组 (A - λᵢ * I) * v = 0 来得到对应的所有特征向量。

**例子：**
考虑矩阵 B = [3 1; 0 3]。
*   求特征值：
    ```
    B - λ * I = [3-λ  1]
                [  0  3-λ]
    det(B - λ * I) = (3-λ)(3-λ) - 1*0 = (3-λ)²
    ```
    解特征方程 (3-λ)² = 0，得到 λ = 3（这是一个二重根）。
*   求特征向量：
    对于 λ = 3，求解 (B - 3 * I) * v = 0：
    ```
    B - 3 * I = [0 1]
                [0 0]
    ```
    方程组为：
    ```
    0*v₁ + 1*v₂ = 0  =>  v₂ = 0
    0*v₁ + 0*v₂ = 0  =>  0 = 0 (恒成立)
    ```
    所以 v₁ 可以是任意值，v₂ = 0。因此，属于特征值 λ = 3 的所有特征向量是形如 [t; 0] (t ≠ 0) 的向量。一个简单的基是 [1; 0]。

## 4. 特征值与特征向量的性质

*   **特征向量的线性组合**：属于同一特征值的特征向量的任意线性组合（只要不全为零）仍然是该特征值的特征向量。
*   **不同特征值的特征向量**：属于不同特征值的特征向量是线性无关的。
*   **复数特征值**：实矩阵的特征值可能是复数。如果矩阵的元素都是实数，那么它的复特征值总是成对出现的（共轭复数对）。
*   **三角矩阵的特征值**：上（下）三角矩阵的特征值就是其主对角线上的元素。
*   **矩阵的迹 (Trace)**：矩阵 A 的迹（主对角线上元素之和）等于其所有特征值之和。
*   **矩阵的行列式**：矩阵 A 的行列式等于其所有特征值的乘积。

## 5. 矩阵对角化 (Diagonalization)

如果一个 n×n 的方阵 A 拥有 n 个线性无关的特征向量，那么 A 可以被**对角化**。

设 A 有 n 个线性无关的特征向量 **v₁, v₂, ..., vₙ**，对应的特征值分别为 λ₁, λ₂, ..., λₙ。

构造一个矩阵 P，其列向量是这些线性无关的特征向量：
```
P = [v₁ v₂ ... vₙ]
```
同时构造一个对角矩阵 D，其对角线元素是对应的特征值：
```
D = diag(λ₁, λ₂, ..., λₙ)
   = [λ₁  0  ...  0 ]
     [ 0  λ₂ ...  0 ]
     [ .   .  .    .]
     [ 0   0  ... λₙ]
```
那么有：
```
A * P = P * D
```
如果 P 可逆（这由特征向量线性无关保证），则：
```
A = P * D * P⁻¹
```
这就是矩阵 A 的**特征分解**或**谱分解 (Spectral Decomposition)**。

**重要性：**
矩阵对角化极大地简化了矩阵的幂运算和函数计算。例如：
```
A^k = (P * D * P⁻¹)^k = P * D^k * P⁻¹
```
而 D^k 是一个对角矩阵，其对角线元素是 λ₁^k, λ₂^k, ..., λₙ^k，计算非常简单。

**可对角化的充要条件：**
*   矩阵 A 有 n 个线性无关的特征向量。
*   对于 A 的每一个特征值，其代数重数（在特征多项式中的重根次数）等于其几何重数（对应特征子空间的维数，即线性无关的特征向量的个数）。
*   拥有 n 个不同特征值的矩阵必然可以对角化。

## 6. 计算特征值与特征向量的数值方法

对于小规模矩阵，可以通过求解特征方程的根来得到特征值。但对于大规模矩阵，符号计算变得不可行且数值不稳定。

### 6.1 幂迭代法 (Power Iteration)

幂迭代法是一种用于计算矩阵**按模最大**特征值及其对应特征向量的简单而有效的数值方法。

**基本思想：**
反复用矩阵 A 乘以一个初始非零向量 **v**，得到序列 **v, Av, A²v, A³v, ...**。在一定条件下，这个序列会收敛到最大特征值对应的特征向量方向。

**算法步骤：**
1.  选择一个初始向量 **v₀**（通常随机选取，且 ||v₀|| ≠ 0）。
2.  对 k = 1, 2, 3, ... 进行迭代：
    a. 计算 **wₖ = A * vₖ₋₁**
    b. 计算 **vₖ = wₖ / ||wₖ||** （归一化）
    c. 估算特征值：**μₖ = vₖᵀ * A * vₖ** （瑞利商）
3.  当 **μₖ** 收敛或达到最大迭代次数时，停止。**μₖ** 是最大特征值的近似，**vₖ** 是对应特征向量的近似。

**局限性：**
*   只能求出按模最大的特征值。
*   如果最大特征值是多重的，或者有多个特征值的模相等，算法可能不收敛。
*   收敛速度取决于最大特征值与第二大特征值的模的比值，比值越大收敛越快。

### 6.2 逆幂迭代法 (Inverse Iteration)

为了求解最小的特征值，可以对矩阵 A 的逆 A⁻¹ 应用幂迭代法。A⁻¹ 的特征值是 1/λ，所以 A⁻¹ 的最大特征值对应于 A 的最小特征值。

### 6.3 QR 算法

QR 算法是一种更复杂的数值方法，可以一次计算出矩阵的**所有**特征值（对于一般矩阵）或所有特征值和特征向量（对于对称矩阵）。它是现代数值线性代数软件包（如 MATLAB, NumPy）中计算特征值的标准方法。其核心思想是通过一系列相似变换（QR 分解）将矩阵逐步转化为近似的上三角形式，对角线上的元素即为特征值。

## 7. 特征值与特征向量在机器学习中的应用

特征值和特征向量在机器学习中有极其广泛的应用：
*   **主成分分析 (PCA)**：通过计算数据协方差矩阵的特征值和特征向量来实现降维，特征向量指示了数据方差最大的方向。
*   **谱聚类 (Spectral Clustering)**：利用图拉普拉斯矩阵的特征值和特征向量进行数据聚类。
*   **奇异值分解 (SVD)**：与特征值分解密切相关，广泛应用于推荐系统、数据压缩和降维。
*   **PageRank 算法**：通过计算一个特殊矩阵的最大特征值对应的特征向量来确定网页的重要性。
*   **稳定性分析**：在动力系统和控制理论中，通过分析系统矩阵的特征值来判断系统的稳定性。

理解特征值和特征向量是掌握这些高级算法和深入理解数据内在结构的关键。
