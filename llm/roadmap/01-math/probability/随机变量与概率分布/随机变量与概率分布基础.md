# 随机变量与概率分布基础

## 1. 引言

在日常生活中，我们经常遇到不确定性事件，例如抛硬币的结果、明天的天气、股票价格的波动等。概率论（Probability Theory）就是研究这些随机现象数量规律的数学分支。

**随机变量**（Random Variable）是连接现实世界随机事件与数学分析的桥梁。它将随机试验的结果映射为数值，使得我们可以用数学工具来分析和处理不确定性。

想象你在玩一个掷骰子的游戏。骰子的点数（1到6）本身就是数值结果，可以直接作为随机变量。但如果游戏规则是“掷出偶数点你赢”，那么我们关心的就不再是具体的点数，而是“输”或“赢”这个结果。我们可以定义一个随机变量 `X`，当掷出偶数时 `X=1`（赢），掷出奇数时 `X=0`（输）。这样，复杂的随机事件就被量化为了数值。

## 2. 随机变量（Random Variable）

### 2.1 定义

随机变量是一个**函数**，它将随机试验的每一个可能结果（样本点）映射到一个实数。

*   通常用大写字母 `X`, `Y`, `Z` 等表示随机变量。
*   随机变量的取值用小写字母 `x`, `y`, `z` 等表示。

### 2.2 类型

随机变量主要分为两大类：

1.  **离散型随机变量（Discrete Random Variable）：**
    *   取值为有限个或可列无限个（可以一一列举出来）。
    *   例子：
        *   掷一次骰子出现的点数 `X ∈ {1, 2, 3, 4, 5, 6}`。
        *   某路口一小时内通过的汽车数量 `Y ∈ {0, 1, 2, 3, ...}`。
        *   抛10次硬币正面朝上的次数 `Z ∈ {0, 1, 2, ..., 10}`。

2.  **连续型随机变量（Continuous Random Variable）：**
    *   取值充满一个区间，无法一一列举。
    *   例子：
        *   某型号灯泡的使用寿命 `T`（可以是 `[0, +∞)` 区间内的任意非负实数）。
        *   某地区成年男性的身高 `H`（可以是 `[0, 3米)` 区间内的任意实数，虽然实际有上限）。
        *   测量某物体长度的误差 `E`（理论上可以是 `(-∞, +∞)` 区间内的任意实数）。

## 3. 概率分布（Probability Distribution）

概率分布描述了随机变量取各个可能值的概率规律。

### 3.1 离散型随机变量的概率分布

对于离散型随机变量 `X`，其所有可能取值为 `x₁, x₂, ...`，`X` 取值为 `xᵢ` 的概率为 `P(X = xᵢ) = pᵢ`。

**概率质量函数（Probability Mass Function, PMF）：**

`p(xᵢ) = P(X = xᵢ)` 称为离散型随机变量 `X` 的概率质量函数。

**性质：**
1.  `p(xᵢ) ≥ 0`，对所有 `i` 成立。
2.  `Σᵢ p(xᵢ) = 1` （所有可能取值的概率之和为1）。

**例子：** 公平骰子
*   随机变量 `X`：掷一次骰子出现的点数。
*   `X` 的取值：`{1, 2, 3, 4, 5, 6}`。
*   概率质量函数：`p(1) = p(2) = ... = p(6) = 1/6`。
*   `p(1) + p(2) + ... + p(6) = 6 * (1/6) = 1`。

### 3.2 连续型随机变量的概率分布

对于连续型随机变量 `X`，由于它取任一具体数值的概率为0（因为点没有“长度”），我们关注的是它落在某个区间内的概率。

**概率密度函数（Probability Density Function, PDF）：**

如果存在一个非负函数 `f(x)`，使得对于任意实数 `a < b`，都有：

```
P(a ≤ X ≤ b) = ∫[a, b] f(x) dx
```

那么 `f(x)` 称为连续型随机变量 `X` 的概率密度函数。

**性质：**
1.  `f(x) ≥ 0`，对所有 `x` 成立。
2.  `∫[-∞, +∞] f(x) dx = 1` （整个实数轴上的“面积”为1）。
3.  `P(X = x) = 0`，对任意 `x` 成立。
4.  `P(a < X ≤ b) = P(a ≤ X < b) = P(a < X < b) = P(a ≤ X ≤ b) = ∫[a, b] f(x) dx` （因为单点概率为0）。

**注意：** `f(x)` 本身**不是**概率，它的值可以大于1。`f(x)` 在某点的“高度”反映了随机变量在该点附近取值的“密集程度”。

**例子：** 均匀分布 `U(a, b)`
*   随机变量 `X` 在区间 `[a, b]` 上均匀分布，记作 `X ~ U(a, b)`。
*   概率密度函数：
    ```
    f(x) = {
        1/(b-a),  a ≤ x ≤ b
        0,        其他
    }
    ```
*   验证性质2：`∫[-∞, +∞] f(x) dx = ∫[a, b] 1/(b-a) dx = [x/(b-a)]|[a, b] = (b-a)/(b-a) = 1`。

## 4. 累积分布函数（Cumulative Distribution Function, CDF）

累积分布函数是一种统一描述离散型和连续型随机变量概率特性的方法。

**定义：**

对于任意随机变量 `X`，其累积分布函数 `F(x)` 定义为：

```
F(x) = P(X ≤ x)
```

即 `F(x)` 表示随机变量 `X` 取值小于或等于 `x` 的概率。

**性质：**
1.  `0 ≤ F(x) ≤ 1`。
2.  `F(x)` 是单调不减函数。
3.  `lim (x -> -∞) F(x) = 0`。
4.  `lim (x -> +∞) F(x) = 1`。
5.  `F(x)` 是右连续的。

### 4.1 离散型随机变量的CDF

如果 `X` 是离散型随机变量，其PMF为 `p(xᵢ)`，则：

```
F(x) = Σ (xᵢ ≤ x) p(xᵢ)
```

即 `F(x)` 是所有小于或等于 `x` 的取值的概率之和。

**例子：** 公平骰子
*   `F(0) = P(X ≤ 0) = 0`
*   `F(1) = P(X ≤ 1) = p(1) = 1/6`
*   `F(2) = P(X ≤ 2) = p(1) + p(2) = 1/6 + 1/6 = 2/6`
*   ...
*   `F(6) = P(X ≤ 6) = 1`
*   `F(7) = P(X ≤ 7) = 1`
*   `F(x)` 是一个阶梯函数。

### 4.2 连续型随机变量的CDF

如果 `X` 是连续型随机变量，其PDF为 `f(x)`，则：

```
F(x) = ∫[-∞, x] f(t) dt
```

根据微积分基本定理，如果 `f(x)` 在 `x` 点连续，则：

```
F'(x) = f(x)
```

即PDF是CDF的导数。

**例子：** 均匀分布 `U(0, 1)`
*   PDF: `f(x) = 1` for `0 ≤ x ≤ 1`, else `0`。
*   CDF:
    *   `x < 0`: `F(x) = ∫[-∞, x] 0 dt = 0`
    *   `0 ≤ x ≤ 1`: `F(x) = ∫[-∞, 0] 0 dt + ∫[0, x] 1 dt = 0 + [t]|[0, x] = x`
    *   `x > 1`: `F(x) = ∫[-∞, 0] 0 dt + ∫[0, 1] 1 dt + ∫[1, x] 0 dt = 0 + 1 + 0 = 1`
    ```
    F(x) = {
        0,     x < 0
        x,     0 ≤ x ≤ 1
        1,     x > 1
    }
    ```

## 5. 总结

*   **随机变量**是将随机试验结果映射为数值的函数，分为离散型和连续型。
*   **概率分布**描述了随机变量取值的规律。
    *   离散型随机变量用**概率质量函数**（PMF）描述。
    *   连续型随机变量用**概率密度函数**（PDF）描述。
*   **累积分布函数**（CDF）`F(x) = P(X ≤ x)` 是一种通用的描述方法，适用于所有类型的随机变量。
    *   离散型：`F(x)` 是阶梯函数。
    *   连续型：`F(x)` 是连续函数，且 `F'(x) = f(x)`。
*   理解随机变量及其分布是后续学习期望、方差、统计推断以及机器学习中概率模型的基础。
