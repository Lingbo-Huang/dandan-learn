# 协方差与相关系数基础

## 1. 引言

在上一节中，我们学习了如何用期望和方差来描述**单个**随机变量的特征。但在实际问题中，我们经常需要研究**两个或多个**随机变量之间的关系。例如，研究广告投入与销售额之间的关系，或者研究气温与冰淇淋销量之间的关系。

**协方差**（Covariance）和**相关系数**（Correlation Coefficient）就是用来量化两个随机变量之间线性关系强度和方向的数字特征。

## 2. 协方差（Covariance）

### 2.1 定义

协方差衡量的是两个随机变量 `X` 和 `Y` 的总体误差（偏离其期望值）的乘积的期望。

**数学定义：**

```
Cov(X, Y) = E[ (X - E(X))(Y - E(Y)) ]
```

### 2.2 计算公式

与方差类似，协方差也有一个更便于计算的公式：

```
Cov(X, Y) = E(XY) - E(X)E(Y)
```

**推导：**
```
Cov(X, Y) = E[ (X - E(X))(Y - E(Y)) ]
          = E[ XY - X*E(Y) - E(X)*Y + E(X)E(Y) ]
          = E[XY] - E[X*E(Y)] - E[E(X)*Y] + E[E(X)E(Y)] // 期望的线性性
          = E[XY] - E(Y)*E[X] - E(X)*E[Y] + E(X)E(Y)    // E(X), E(Y)是常数
          = E[XY] - E(X)E(Y)
```

### 2.3 性质

协方差具有以下重要性质（设 `X`, `Y`, `Z` 是随机变量，`a`, `b` 是常数）：

1.  **对称性：** `Cov(X, Y) = Cov(Y, X)`
2.  **与方差的关系：** `Cov(X, X) = Var(X)`
3.  **常数：** `Cov(X, c) = 0`
4.  **线性性：**
    *   `Cov(aX, Y) = a * Cov(X, Y)`
    *   `Cov(X + Z, Y) = Cov(X, Y) + Cov(Z, Y)`
    *   推广：`Cov(aX + bY, Z) = a * Cov(X, Z) + b * Cov(Y, Z)`
5.  **独立性：** 如果 `X` 和 `Y` **相互独立**，则 `Cov(X, Y) = 0`。
    *   **注意：** `Cov(X, Y) = 0` 并**不能**推出 `X` 和 `Y` 独立。它只能说明没有**线性**关系，可能存在其他非线性关系。

### 2.4 解释

*   **`Cov(X, Y) > 0`：** 趋势相同。当 `X` 大于其期望时，`Y` 也倾向于大于其期望；当 `X` 小于其期望时，`Y` 也倾向于小于其期望。称 `X` 与 `Y` **正相关**。
*   **`Cov(X, Y) < 0`：** 趋势相反。当 `X` 大于其期望时，`Y` 倾向于小于其期望；反之亦然。称 `X` 与 `Y` **负相关**。
*   **`Cov(X, Y) = 0`：** 没有线性趋势。称 `X` 与 `Y` **不相关**（或线性无关）。

**局限性：**
协方差的值受变量本身量纲的影响。例如，`X` 以米为单位，`Y` 以千克为单位，那么 `Cov(X, Y)` 的单位就是 米*千克，这使得比较不同变量对之间的相关性强弱变得困难。

## 3. 相关系数（Correlation Coefficient）

为了解决协方差量纲不统一的问题，我们引入了相关系数。

### 3.1 定义（皮尔逊相关系数）

**皮尔逊相关系数**（Pearson Correlation Coefficient）是协方差除以两个变量标准差的乘积。

```
ρ_XY = Corr(X, Y) = Cov(X, Y) / (σ_X * σ_Y)
```

其中 `σ_X = √Var(X)` 和 `σ_Y = √Var(Y)` 分别是 `X` 和 `Y` 的标准差。

### 3.2 性质

1.  **取值范围：** `-1 ≤ ρ_XY ≤ 1`
2.  **无量纲：** `ρ_XY` 是一个纯数，没有单位。
3.  **对称性：** `ρ_XY = ρ_YX`
4.  **不变性：** 相关系数不受变量平移和尺度变换的影响。
    *   `Corr(aX + b, cY + d) = sign(ac) * Corr(X, Y)` （其中 `a, c ≠ 0`）
5.  **独立性：** 如果 `X` 和 `Y` 独立，则 `ρ_XY = 0`。
6.  **完全线性关系：**
    *   `ρ_XY = 1` 当且仅当存在常数 `a > 0` 和 `b` 使得 `Y = aX + b` （完全正线性相关）。
    *   `ρ_XY = -1` 当且仅当存在常数 `a < 0` 和 `b` 使得 `Y = aX + b` （完全负线性相关）。

### 3.3 解释

*   **`ρ_XY = 1`：** 完全正线性相关。所有点 `(X, Y)` 都在一条斜率为正的直线上。
*   **`ρ_XY = -1`：** 完全负线性相关。所有点 `(X, Y)` 都在一条斜率为负的直线上。
*   **`ρ_XY = 0`：** 没有线性相关性。但这不意味着 `X` 和 `Y` 没有任何关系，可能只是没有线性关系。
*   **`0 < |ρ_XY| < 1`：** 存在一定程度的线性相关性。
    *   `|ρ_XY|` 越接近1，线性相关性越强。
    *   `|ρ_XY|` 越接近0，线性相关性越弱。

**经验判断：**
*   `|ρ| ≥ 0.8`：强相关
*   `0.5 ≤ |ρ| < 0.8`：中等相关
*   `0.3 ≤ |ρ| < 0.5`：弱相关
*   `|ρ| < 0.3`：极弱相关或无相关

### 3.4 计算示例

**例子：** 设二维随机变量 `(X, Y)` 的联合分布律如下：

| X\Y | 1   | 2   | 3   |
| :-- | :-- | :-- | :-- |
| 1   | 0.1 | 0.2 | 0.1 |
| 2   | 0.1 | 0.2 | 0.3 |

求 `Cov(X, Y)` 和 `ρ_XY`。

**步骤1：计算边缘分布和期望**

*   `X` 的边缘分布：
    *   `P(X=1) = 0.1 + 0.2 + 0.1 = 0.4`
    *   `P(X=2) = 0.1 + 0.2 + 0.3 = 0.6`
*   `Y` 的边缘分布：
    *   `P(Y=1) = 0.1 + 0.1 = 0.2`
    *   `P(Y=2) = 0.2 + 0.2 = 0.4`
    *   `P(Y=3) = 0.1 + 0.3 = 0.4`
*   `E(X) = 1*0.4 + 2*0.6 = 1.6`
*   `E(Y) = 1*0.2 + 2*0.4 + 3*0.4 = 2.2`

**步骤2：计算 `E(XY)`**

*   `E(XY) = Σᵢ Σⱼ xᵢ yⱼ P(X=xᵢ, Y=yⱼ)`
*   `E(XY) = 1*1*0.1 + 1*2*0.2 + 1*3*0.1 + 2*1*0.1 + 2*2*0.2 + 2*3*0.3`
*   `E(XY) = 0.1 + 0.4 + 0.3 + 0.2 + 0.8 + 1.8 = 3.6`

**步骤3：计算协方差**

*   `Cov(X, Y) = E(XY) - E(X)E(Y) = 3.6 - 1.6*2.2 = 3.6 - 3.52 = 0.08`

**步骤4：计算方差和标准差**

*   `E(X²) = 1²*0.4 + 2²*0.6 = 0.4 + 2.4 = 2.8`
*   `Var(X) = E(X²) - [E(X)]² = 2.8 - (1.6)² = 2.8 - 2.56 = 0.24`
*   `σ_X = √0.24 ≈ 0.4899`
*   `E(Y²) = 1²*0.2 + 2²*0.4 + 3²*0.4 = 0.2 + 1.6 + 3.6 = 5.4`
*   `Var(Y) = E(Y²) - [E(Y)]² = 5.4 - (2.2)² = 5.4 - 4.84 = 0.56`
*   `σ_Y = √0.56 ≈ 0.7483`

**步骤5：计算相关系数**

*   `ρ_XY = Cov(X, Y) / (σ_X * σ_Y) = 0.08 / (0.4899 * 0.7483) ≈ 0.08 / 0.3666 ≈ 0.218`
*   解释：`X` 和 `Y` 之间存在微弱的正线性相关性。

## 4. 总结

*   **协方差** `Cov(X, Y)` 衡量两个随机变量的线性相关方向。
    *   `> 0`：正相关；`< 0`：负相关；`= 0`：不相关（无线性关系）。
    *   受变量量纲影响，难以比较不同变量对的相关性强弱。
*   **相关系数** `ρ_XY` 是标准化的协方差，消除了量纲影响。
    *   取值范围 `[-1, 1]`。
    *   `1`：完全正线性相关；`-1`：完全负线性相关；`0`：无线性相关。
    *   `|ρ|` 越接近1，线性相关性越强。
*   **重要提醒：**
    *   相关系数只衡量**线性**相关性。即使 `ρ = 0`，`X` 和 `Y` 之间也可能存在非线性关系。
    *   相关性**不等于**因果性。`X` 和 `Y` 高度相关，并不意味着 `X` 导致 `Y` 或 `Y` 导致 `X`。
*   协方差和相关系数是多元统计分析、回归分析以及机器学习中特征选择和模型理解的基础。
