# 常见概率分布基础

## 1. 引言

在概率论和统计学中，有一些特定的概率分布因其在建模现实世界现象时的普遍性和重要性而被广泛研究和应用。理解这些**常见概率分布**的性质、期望、方差以及它们适用的场景，对于数据分析、统计推断和机器学习至关重要。

就像在工具箱里有不同用途的工具一样，不同的概率分布适用于描述不同类型的数据生成过程。例如，二项分布适合描述固定次数独立试验中成功次数的分布，而泊松分布适合描述单位时间内稀有事件发生次数的分布。

## 2. 离散型概率分布

### 2.1 伯努利分布（Bernoulli Distribution）

**背景：** 描述一次只有两种可能结果（成功/失败，是/否，1/0）的随机试验。

**参数：** 成功概率 `p` (0 < p < 1)

**记法：** `X ~ B(1, p)` 或 `X ~ Bernoulli(p)`

**概率质量函数（PMF）：**
```
P(X = k) = {
    p,      k = 1 (成功)
    1-p,    k = 0 (失败)
}
```
或者简写为：`P(X = k) = p^k * (1-p)^(1-k)`, `k ∈ {0, 1}`

**期望与方差：**
*   `E(X) = p`
*   `Var(X) = p(1-p)`

**例子：** 抛一次硬币，正面朝上（成功）的概率为 `p=0.6`。

### 2.2 二项分布（Binomial Distribution）

**背景：** 描述 `n` 次独立的伯努利试验中成功次数的分布。

**参数：** 试验次数 `n` (正整数) 和单次成功概率 `p` (0 < p < 1)

**记法：** `X ~ B(n, p)`

**概率质量函数（PMF）：**
```
P(X = k) = C(n, k) * p^k * (1-p)^(n-k),  k = 0, 1, 2, ..., n
```
其中 `C(n, k) = n! / (k!(n-k)!)` 是组合数，表示从 `n` 次试验中选出 `k` 次成功的方法数。

**期望与方差：**
*   `E(X) = np`
*   `Var(X) = np(1-p)`

**例子：** 抛10次硬币，正面朝上的次数 `X` 服从 `B(10, 0.5)`。

### 2.3 泊松分布（Poisson Distribution）

**背景：** 描述单位时间（或空间）内稀有事件发生次数的分布。当二项分布的 `n` 很大而 `p` 很小时，二项分布近似于泊松分布，其中 `λ = np`。

**参数：** 平均发生率 `λ` (λ > 0)

**记法：** `X ~ P(λ)` 或 `X ~ Poisson(λ)`

**概率质量函数（PMF）：**
```
P(X = k) = (λ^k * e^(-λ)) / k!,  k = 0, 1, 2, ...
```

**期望与方差：**
*   `E(X) = λ`
*   `Var(X) = λ`

**例子：**
*   一小时内到达某服务台的顾客数。
*   一页书中印刷错误的个数。
*   某放射性物质在1秒内衰变的原子核数。

### 2.4 几何分布（Geometric Distribution）

**背景：** 描述在独立的伯努利试验中，首次成功所需要的试验次数。

**参数：** 单次成功概率 `p` (0 < p < 1)

**记法：** `X ~ Geom(p)`

**概率质量函数（PMF）：**
*   **形式1（试验次数）：** `X` 表示首次成功发生在第 `k` 次试验。
    ```
    P(X = k) = (1-p)^(k-1) * p,  k = 1, 2, 3, ...
    ```
*   **形式2（失败次数）：** `Y` 表示首次成功前的失败次数。
    ```
    P(Y = k) = (1-p)^k * p,  k = 0, 1, 2, ...
    ```

**期望与方差（以形式1为例）：**
*   `E(X) = 1/p`
*   `Var(X) = (1-p) / p²`

**例子：** 抛硬币直到第一次出现正面，抛掷的次数。

### 2.5 超几何分布（Hypergeometric Distribution）

**背景：** 描述不放回抽样中成功个体数的分布。与二项分布（有放回抽样）相对。

**参数：**
*   总体大小 `N`
*   总体中成功个体数 `K`
*   抽取的个体数 `n`

**记法：** `X ~ H(N, K, n)`

**概率质量函数（PMF）：**
```
P(X = k) = [C(K, k) * C(N-K, n-k)] / C(N, n)
```
其中 `max(0, n-K+N) ≤ k ≤ min(K, n)`

**期望与方差：**
*   `E(X) = n * (K/N)`
*   `Var(X) = n * (K/N) * (1 - K/N) * (N-n)/(N-1)`
    *   `(N-n)/(N-1)` 是**超几何修正因子**（Finite Population Correction）。

**例子：** 从一副52张的扑克牌中不放回地抽取5张，抽到黑桃的张数。

## 3. 连续型概率分布

### 3.1 均匀分布（Uniform Distribution）

**背景：** 描述在区间 `[a, b]` 内任意一点被选中的概率相等的情况。

**参数：** 区间端点 `a` 和 `b` (a < b)

**记法：** `X ~ U(a, b)`

**概率密度函数（PDF）：**
```
f(x) = {
    1/(b-a),  a ≤ x ≤ b
    0,        其他
}
```

**累积分布函数（CDF）：**
```
F(x) = {
    0,           x < a
    (x-a)/(b-a), a ≤ x ≤ b
    1,           x > b
}
```

**期望与方差：**
*   `E(X) = (a+b)/2`
*   `Var(X) = (b-a)²/12`

**例子：** 理想的随机数生成器在 `[0, 1]` 区间内生成的数字。

### 3.2 指数分布（Exponential Distribution）

**背景：** 描述事件之间时间间隔的分布，常用于描述寿命或等待时间。它是泊松过程的等待时间分布。

**参数：** 速率参数 `λ` (λ > 0)。有时也用尺度参数 `β = 1/λ`。

**记法：** `X ~ Exp(λ)`

**概率密度函数（PDF）：**
```
f(x) = {
    λ * e^(-λx),  x ≥ 0
    0,            x < 0
}
```

**累积分布函数（CDF）：**
```
F(x) = {
    1 - e^(-λx),  x ≥ 0
    0,            x < 0
}
```

**期望与方差：**
*   `E(X) = 1/λ`
*   `Var(X) = 1/λ²`

**无记忆性（Memoryless Property）：**
```
P(X > s + t | X > s) = P(X > t)
```
这表示，已知事件在时间 `s` 后仍未发生，它在之后再等待时间 `t` 的概率，等于从一开始等待时间 `t` 的概率。例如，一个寿命服从指数分布的元件，使用了 `s` 小时后，它还能再使用 `t` 小时的概率，和一个全新的元件能使用 `t` 小时的概率相同。

**例子：**
*   顾客到达服务台的时间间隔。
*   电子元件的寿命。
*   放射性原子核衰变的时间间隔。

### 3.3 正态分布（Normal Distribution / Gaussian Distribution）

**背景：** 最重要和最常见的连续型分布。大量独立随机因素的和近似服从正态分布（中心极限定理）。自然界和社会科学中的许多现象都近似服从正态分布。

**参数：** 均值 `μ` (实数) 和方差 `σ²` (σ² > 0)。标准差为 `σ`。

**记法：** `X ~ N(μ, σ²)`

**概率密度函数（PDF）：**
```
f(x) = (1 / (σ * √(2π))) * e^(- (x-μ)² / (2σ²))
```

**性质：**
*   图形是关于 `x = μ` 对称的钟形曲线。
*   `μ` 决定曲线的中心位置。
*   `σ` 决定曲线的宽窄程度（越小越陡峭）。
*   **标准正态分布：** 当 `μ=0`, `σ=1` 时，称为标准正态分布，记作 `Z ~ N(0, 1)`。其PDF为 `φ(z) = (1/√(2π)) * e^(-z²/2)`。

**期望与方差：**
*   `E(X) = μ`
*   `Var(X) = σ²`

**标准化：** 任何正态分布 `X ~ N(μ, σ²)` 都可以通过线性变换标准化为标准正态分布：
```
Z = (X - μ) / σ ~ N(0, 1)
```

**68-95-99.7 规则（经验法则）：**
对于 `X ~ N(μ, σ²)`：
*   `P(μ-σ < X < μ+σ) ≈ 68.27%`
*   `P(μ-2σ < X < μ+2σ) ≈ 95.45%`
*   `P(μ-3σ < X < μ+3σ) ≈ 99.73%`

**例子：**
*   人的身高、体重。
*   测量误差。
*   考试成绩（在某些情况下）。

### 3.4 伽马分布（Gamma Distribution）

**背景：** 指数分布和卡方分布（Chi-Square Distribution）是其特例。常用于排队论和可靠性分析。

**参数：** 形状参数 `α` (α > 0) 和 速率参数 `β` (β > 0)。有时也用尺度参数 `θ = 1/β`。

**记法：** `X ~ Gamma(α, β)`

**概率密度函数（PDF）：**
```
f(x) = {
    (β^α / Γ(α)) * x^(α-1) * e^(-βx),  x > 0
    0,                                   x ≤ 0
}
```
其中 `Γ(α) = ∫[0, +∞] t^(α-1) * e^(-t) dt` 是伽马函数。

**期望与方差：**
*   `E(X) = α/β`
*   `Var(X) = α/β²`

**特例：**
*   当 `α=1` 时，`Gamma(1, β)` 就是 `Exp(β)` 指数分布。
*   当 `α = n/2` 且 `β = 1/2` 时，`Gamma(n/2, 1/2)` 就是自由度为 `n` 的卡方分布 `χ²(n)`。

### 3.5 贝塔分布（Beta Distribution）

**背景：** 定义在 `[0, 1]` 区间上，常用于建模概率或比例。在贝叶斯统计中，它是伯努利分布和二项分布的共轭先验。

**参数：** 形状参数 `α` (α > 0) 和 `β` (β > 0)

**记法：** `X ~ Beta(α, β)`

**概率密度函数（PDF）：**
```
f(x) = {
    (Γ(α+β) / (Γ(α)Γ(β))) * x^(α-1) * (1-x)^(β-1),  0 < x < 1
    0,                                               其他
}
```

**期望与方差：**
*   `E(X) = α / (α+β)`
*   `Var(X) = (αβ) / [(α+β)²(α+β+1)]`

## 4. 总结

*   **离散型分布**：伯努利（单次试验）、二项（n次试验成功数）、泊松（单位时间事件数）、几何（首次成功试验次数）、超几何（不放回抽样成功数）。
*   **连续型分布**：均匀（区间内等概率）、指数（事件间隔时间）、正态（自然界最常见）、伽马（指数分布推广）、贝塔（[0,1]区间概率）。
*   每种分布都有其特定的背景、参数、概率函数、期望和方差。
*   理解这些分布有助于：
    1.  为实际问题选择合适的概率模型。
    2.  进行统计推断（参数估计、假设检验）。
    3.  在机器学习中进行概率建模和贝叶斯分析。
