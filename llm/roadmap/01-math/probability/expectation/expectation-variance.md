# 期望与方差基础

## 1. 引言

随机变量的概率分布描述了它所有可能取值及其对应的概率。但在实际应用中，我们往往希望用一个或几个**数值**来概括随机变量的主要特征。**期望**（Expectation）和**方差**（Variance）就是两个最重要的数字特征。

*   **期望**（或均值）描述了随机变量取值的**平均水平**或**中心位置**。
*   **方差**描述了随机变量取值相对于其期望的**离散程度**或**波动大小**。

想象你在分析一个班级学生的考试成绩。成绩是一个随机变量。期望就是平均分，告诉我们班级整体水平；方差则告诉我们分数的分布情况，方差大说明学生之间分数差距大，方差小说明分数比较集中。

## 2. 期望（Expectation / Mean）

### 2.1 定义

期望是随机变量所有可能取值与其对应概率乘积的总和（对于离散型）或积分（对于连续型）。

#### 离散型随机变量的期望

设离散型随机变量 `X` 的概率质量函数为 `p(xᵢ)`，则 `X` 的期望定义为：

```
E(X) = Σᵢ xᵢ * p(xᵢ)
```

前提是级数绝对收敛。

#### 连续型随机变量的期望

设连续型随机变量 `X` 的概率密度函数为 `f(x)`，则 `X` 的期望定义为：

```
E(X) = ∫[-∞, +∞] x * f(x) dx
```

前提是积分绝对收敛。

### 2.2 性质

期望具有以下重要性质（设 `X`, `Y` 是随机变量，`c` 是常数）：

1.  **常数的期望：** `E(c) = c`
2.  **线性性：**
    *   `E(cX) = c * E(X)`
    *   `E(X + Y) = E(X) + E(Y)`
    *   推广：`E(aX + bY + c) = a * E(X) + b * E(Y) + c`
3.  **函数的期望：** 如果 `Y = g(X)` 是 `X` 的函数，那么：
    *   离散型：`E(Y) = E[g(X)] = Σᵢ g(xᵢ) * p(xᵢ)`
    *   连续型：`E(Y) = E[g(X)] = ∫[-∞, +∞] g(x) * f(x) dx`

**注意：** 一般情况下，`E[g(X)] ≠ g[E(X)]`。例如，`E[X²] ≠ [E(X)]²`。

### 2.3 计算示例

**例子1：** 掷一次公平骰子，求点数 `X` 的期望。

*   `X` 的取值：`{1, 2, 3, 4, 5, 6}`，每个值的概率都是 `1/6`。
*   `E(X) = Σ (i=1 to 6) i * P(X=i) = 1*(1/6) + 2*(1/6) + ... + 6*(1/6) = (1+2+3+4+5+6)/6 = 21/6 = 3.5`
*   解释：虽然骰子没有3.5点，但长期来看，大量重复掷骰子得到的点数平均值会趋近于3.5。

**例子2：** 设连续型随机变量 `X` 服从 `[0, 2]` 上的均匀分布，求 `E(X)`。

*   PDF: `f(x) = 1/2` for `0 ≤ x ≤ 2`, else `0`。
*   `E(X) = ∫[-∞, +∞] x * f(x) dx = ∫[0, 2] x * (1/2) dx = (1/2) * ∫[0, 2] x dx = (1/2) * [x²/2]|[0, 2] = (1/2) * (4/2 - 0) = (1/2) * 2 = 1`
*   解释：均匀分布在区间中点处取得期望值。

## 3. 方差（Variance）

### 3.1 定义

方差衡量的是随机变量 `X` 的取值与其期望 `E(X)` 的偏离程度的平方的平均值。

```
Var(X) = D(X) = E[ (X - E(X))² ]
```

**标准差（Standard Deviation）** 是方差的平方根，与原变量有相同的量纲：

```
σ_X = √Var(X)
```

### 3.2 计算公式

直接按定义计算方差通常比较麻烦。有一个更常用的计算公式：

```
Var(X) = E[X²] - [E(X)]²
```

**推导：**
```
Var(X) = E[ (X - E(X))² ]
       = E[ X² - 2X*E(X) + (E(X))² ]
       = E[X²] - E[2X*E(X)] + E[(E(X))²]  // 利用期望的线性性
       = E[X²] - 2*E(X)*E(X) + (E(X))²   // E(X)是常数
       = E[X²] - 2(E(X))² + (E(X))²
       = E[X²] - (E(X))²
```

### 3.3 性质

方差具有以下重要性质（设 `X`, `Y` 是随机变量，`c` 是常数）：

1.  **常数的方差：** `Var(c) = 0`
2.  **常数因子：** `Var(cX) = c² * Var(X)`
3.  **平移不变性：** `Var(X + c) = Var(X)` （加常数不改变方差）
4.  **独立随机变量的可加性：** 如果 `X` 和 `Y` **相互独立**，则：
    *   `Var(X + Y) = Var(X) + Var(Y)`
    *   `Var(X - Y) = Var(X) + Var(Y)`

### 3.4 计算示例

**例子1：** （续掷骰子例子）求点数 `X` 的方差。

*   已知 `E(X) = 3.5`。
*   先计算 `E[X²]`：
    *   `E[X²] = Σ (i=1 to 6) i² * P(X=i) = 1²*(1/6) + 2²*(1/6) + ... + 6²*(1/6) = (1+4+9+16+25+36)/6 = 91/6`
*   应用公式：`Var(X) = E[X²] - [E(X)]² = 91/6 - (3.5)² = 91/6 - 12.25 = 91/6 - 49/4 = (182 - 147)/12 = 35/12 ≈ 2.92`
*   标准差：`σ_X = √(35/12) ≈ 1.71`

**例子2：** （续均匀分布例子）求 `X ~ U(0, 2)` 的方差。

*   已知 `E(X) = 1`。
*   先计算 `E[X²]`：
    *   `E[X²] = ∫[-∞, +∞] x² * f(x) dx = ∫[0, 2] x² * (1/2) dx = (1/2) * ∫[0, 2] x² dx = (1/2) * [x³/3]|[0, 2] = (1/2) * (8/3) = 4/3`
*   应用公式：`Var(X) = E[X²] - [E(X)]² = 4/3 - 1² = 4/3 - 1 = 1/3`
*   标准差：`σ_X = √(1/3) ≈ 0.58`

## 4. 常见分布的期望与方差

记住一些常用分布的期望和方差很有帮助。

1.  **伯努利分布（Bernoulli Distribution）** `X ~ B(1, p)`：
    *   `E(X) = p`
    *   `Var(X) = p(1-p)`

2.  **二项分布（Binomial Distribution）** `X ~ B(n, p)`：
    *   `E(X) = np`
    *   `Var(X) = np(1-p)`

3.  **泊松分布（Poisson Distribution）** `X ~ P(λ)`：
    *   `E(X) = λ`
    *   `Var(X) = λ`

4.  **均匀分布（Uniform Distribution）** `X ~ U(a, b)`：
    *   `E(X) = (a+b)/2`
    *   `Var(X) = (b-a)²/12`

5.  **指数分布（Exponential Distribution）** `X ~ Exp(λ)`：
    *   `E(X) = 1/λ`
    *   `Var(X) = 1/λ²`

6.  **正态分布（Normal Distribution）** `X ~ N(μ, σ²)`：
    *   `E(X) = μ`
    *   `Var(X) = σ²`

## 5. 协方差与相关系数（预览）

虽然我们将在下一节详细讨论，但在这里提一下：期望和方差可以推广到描述**两个**随机变量之间的关系。

*   **协方差** `Cov(X, Y) = E[ (X - E(X))(Y - E(Y)) ]` 衡量两个变量的总体误差的乘积的期望。它描述了两个变量的**线性相关方向**。
*   **相关系数** `ρ_XY = Cov(X, Y) / (σ_X * σ_Y)` 是标准化的协方差，取值范围在 `[-1, 1]` 之间，衡量两个变量的**线性相关强度**。

## 6. 总结

*   **期望** `E(X)` 是随机变量的平均取值，描述其中心位置。
*   **方差** `Var(X)` 是随机变量取值偏离其期望的平方的平均值，描述其离散程度。标准差 `σ_X` 与其有相同量纲。
*   计算方差的常用公式：`Var(X) = E[X²] - [E(X)]²`。
*   期望具有线性性，方差在独立条件下具有可加性。
*   期望和方差是随机变量最重要的数字特征，是后续学习概率论和数理统计（如大数定律、中心极限定理）以及机器学习（如模型评估）的基础。
