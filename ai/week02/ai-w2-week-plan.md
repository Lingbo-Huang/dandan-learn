## 知识图谱

```
矩阵 A
 ├── 方阵
 │    ├── 特征值 λ  ──── det(A - λI) = 0
 │    ├── 特征向量 v ──── (A - λI)v = 0
 │    └── 可对角化？
 │         ├── Yes → A = PΛP⁻¹  （谱分解）
 │         └── No  → Jordan 标准型（本课不展开）
 │
 └── 任意矩阵（m×n）
      └── SVD：A = UΣVᵀ
           ├── U  = A 的左奇异向量（列空间基）
           ├── Σ  = 奇异值（非负实数，降序）
           ├── V  = A 的右奇异向量（行空间基）
           └── PCA = 对协方差矩阵做 SVD / 特征分解
```
