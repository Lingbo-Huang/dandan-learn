# AI 基础线 Week 1 周规划总览

**主题：线性代数①——向量、矩阵、行列式、矩阵乘法**

> 目标：打牢 AI/ML 所需的线性代数基础，能够手算+代码验证核心运算，为后续神经网络、优化算法做铺垫。
> 工具：Python + `uv`（不使用 conda）

---

## 每日一览

| 天 | 主题 | 关键词 |
|---|---|---|
| D1 | 向量基础 | 向量定义、加减法、数乘、点积、范数 |
| D2 | 向量进阶 | 线性无关、基、向量空间、投影、余弦相似度 |
| D3 | 矩阵基础 | 矩阵定义、加减法、数乘、转置、对称矩阵 |
| D4 | 矩阵乘法 | 矩阵乘法规则、几何直觉、矩阵链式法则 |
| D5 | 行列式 | 行列式定义、2×2/3×3计算、几何意义、性质 |
| D6 | 矩阵的逆与秩 | 逆矩阵、伴随矩阵法、高斯消元、矩阵秩 |
| D7 | 综合复习与实战 | NumPy 综合实战、线性方程组求解、知识点串联 |

---

## 学习目标（Week 1 结束时）

- [ ] 掌握向量的基本运算，理解点积与投影的几何含义
- [ ] 能用手算和 NumPy 完成矩阵加减、乘法、转置
- [ ] 理解行列式的几何意义（面积/体积缩放），能计算 2×2、3×3 行列式
- [ ] 理解逆矩阵存在条件，能用 NumPy 求逆
- [ ] 能用线性代数工具解线性方程组 Ax = b

## 环境搭建（仅需一次）

```bash
# 安装 uv（若未安装）
curl -Ls https://astral.sh/uv/install.sh | sh

# 创建项目虚拟环境
cd ~/dandan-learn/ai
uv init linalg-week01
cd linalg-week01
uv add numpy matplotlib sympy jupyter

# 启动 Jupyter
uv run jupyter notebook
```
