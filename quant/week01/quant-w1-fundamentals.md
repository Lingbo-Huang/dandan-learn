## 📚 核心知识点

### 1. 期货的本质

**定义：** 在未来某一时刻、以今天约定的价格买卖标的物的标准化合约。

**四大核心特征：**

| 特征 | 说明 |
|------|------|
| **标准化** | 合约规格（数量、质量、交割地点）由交易所统一规定 |
| **杠杆** | 只需缴纳合约价值的5%~15%作为保证金即可交易 |
| **双向** | 可做多（买涨）也可做空（买跌），T+0 |
| **交割** | 到期必须进行实物或现金交割（大多数投机者提前平仓）|

**期货 vs 股票：**
```
股票：买入持有，享受分红，无到期日
期货：杠杆工具，有到期日，到期要交割或平仓
```

### 2. 国内期货交易所

| 交易所 | 简称 | 核心品种 |
|--------|------|----------|
| 上海期货交易所 | 上期所 SHFE | 铜、铝、锌、金、银、螺纹钢、原油 |
| 大连商品交易所 | 大商所 DCE | 铁矿石、焦炭、焦煤、豆粕、玉米、塑料 |
| 郑州商品交易所 | 郑商所 CZCE | 白糖、棉花、菜粕、苹果、PTA、尿素 |
| 上海国际能源交易中心 | 能源中心 INE | 原油（SC）、低硫燃料油（LU）|
| 中国金融期货交易所 | 中金所 CFFEX | 沪深300股指期货（IF）、国债期货（T）|

### 3. 合约规格示例

以**螺纹钢（RB）**为例：
```
交易单位：10吨/手
报价单位：元/吨
最小变动价位：1元/吨（即每跳 = 10元/手）
涨跌停板：上一交易日结算价的 ±4%
交割方式：实物交割
合约月份：当月起连续3个月+其后的2个双月
```

以**沪深300股指期货（IF）**为例：
```
交易单位：300元/点 × 合约价值
最小变动价位：0.2点（每跳 = 60元）
涨跌停：上一交易日结算价的 ±10%
交割方式：现金交割（按最后交易日沪深300指数均值）
交易时间：09:30-11:30 / 13:00-15:00（比股票早开15分钟）
保证金：约12%（IF）
```

### 4. 关键概念

**主力合约（连续合约）：**
- 每个品种同时有多个月份合约在交易
- 成交量/持仓量最大的合约称为"主力合约"
- 量化回测通常用"主力连续合约"（自动换月拼接）

**基差（Basis）：**
```
基差 = 现货价格 - 期货价格
  正基差（现货溢价）：供需紧张，期货被低估
  负基差（期货溢价）：市场预期价格上涨，反向则期货升水
```

**保证金与爆仓：**
```
假设螺纹钢当前价格 4000元/吨，每手=10吨
  合约面值 = 4000 × 10 = 40,000元
  保证金率 = 10%
  需缴保证金 = 4,000元 持有 1手

当合约亏损导致账户保证金低于维持保证金时→追加保证金
继续亏损无法追缴→强制平仓（爆仓）
```

**升水与贴水：**
- **升水（Contango）**：远月合约价格 > 近月合约价格（市场预期上涨/持仓成本高）
- **贴水（Backwardation）**：远月合约价格 < 近月合约价格（现货紧缺/预期价格回落）

### 5. 市场参与者

| 类型 | 目的 | 典型行为 |
|------|------|----------|
| **套期保值者** | 对冲现货风险 | 航空公司买入原油期货锁定成本 |
| **投机者** | 获取价差利润 | 日内交易、趋势跟踪 |
| **套利者** | 利用价差无风险获利 | 跨期套利、跨市套利 |

### 6. 国际期货市场

| 交易所 | 代表品种 |
|--------|---------|
| CME（芝加哥商业交易所）| 原油(CL)、黄金(GC)、标普500期货(ES)、欧元期货 |
| NYMEX | 原油、天然气 |
| COMEX | 黄金、白银、铜 |

```python

```

### 计算螺纹钢的波动特征

```python
import numpy as np

# 计算日收益率
df_rb['returns'] = df_rb['close'].pct_change()

# 年化波动率
ann_vol = df_rb['returns'].std() * np.sqrt(252)
print(f"年化波动率：{ann_vol:.2%}")

# 最大回撤
roll_max = df_rb['close'].cummax()
drawdown = (df_rb['close'] - roll_max) / roll_max
print(f"最大回撤：{drawdown.min():.2%}")
```

## 📝 小结

**今天学到了什么：**

1. **期货是杠杆工具**，用少量保证金控制大额合约，收益和风险都被放大
2. **期货 T+0 双向**，可以做多也可以做空，策略灵活度远超股票
3. **主力合约是量化的核心标的**，需要理解换月规则，构建连续合约
4. **基差是套利的来源**，现货与期货的价差包含持仓成本和市场预期
5. **黑色系、能化系品种高度联动**，产业链逻辑是跨品种策略的基础

**关键术语：**
- 多头/空头：买方（期待涨价）/卖方（期待跌价）
- 保证金：期货交易需要缴纳的履约担保金
- 基差 = 现货 - 期货（可正可负）
- 升水（Contango）/ 贴水（Backwardation）
- 主力合约：当前成交量最大的合约月份
