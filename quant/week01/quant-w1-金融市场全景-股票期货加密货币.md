## 📚 核心知识点

### 1. 市场结构

| 交易所 | 简称 | 定位 | 典型上市品种 |
|--------|------|------|------------|
| 上海证券交易所 | 上交所 / SSE | 蓝筹、国企、科创 | 工商银行、贵州茅台、科创板 |
| 深圳证券交易所 | 深交所 / SZSE | 成长、创业 | 创业板、中小板（已并入主板）|
| 北京证券交易所 | 北交所 / BSE | 专精特新中小企业 | 创新层转板企业 |

**板块划分：**
- **主板**：上交所主板 + 深交所主板（传统大盘蓝筹）
- **科创板**（上交所）：硬科技企业，注册制，±20% 涨跌停
- **创业板**（深交所）：成长型企业，注册制，±20% 涨跌停
- **北交所**：中小创新企业，±30% 涨跌停

### 2. 交易规则

**时间安排：**
```
集合竞价：09:15 - 09:25（开盘价形成）
连续竞价：09:30 - 11:30 / 13:00 - 15:00
盘后固定价格交易：15:05 - 15:30
```

**核心规则：**
- **T+1**：当天买入的股票，次日才能卖出
- **涨跌停**：主板 ±10%，科创板/创业板 ±20%（ST股 ±5%）
- **最小交易单位**：1手 = 100股
- **印花税**：卖出时收取 0.1%（2023年起，买入免征）
- **过户费**：0.002%（双向）

**竞价制度：**
- 开盘集合竞价：价格优先 + 时间优先
- 盘中连续竞价：撮合成交，可限价单/市价单

### 3. 指数体系

| 指数 | 成分股数量 | 特点 |
|------|-----------|------|
| 上证指数（000001） | ~2200只 | A股最知名指数，含债券，代表性偏大盘 |
| 深证成指（399001） | 500只 | 深交所代表性指数 |
| 沪深300（000300） | 300只 | 覆盖沪深两市大盘，机构基准 |
| 中证500（000905） | 500只 | 中小盘代表，沪深300之外 |
| 中证1000（000852） | 1000只 | 小盘代表 |
| 创业板指（399006） | 100只 | 创业板龙头 |
| 科创50（000688） | 50只 | 科创板代表 |

### 4. 股票代码规则

```
上交所：6xxxxx（如 600519 贵州茅台）
深交所主板：000xxx / 001xxx（如 000858 五粮液）
创业板：300xxx / 301xxx（如 300750 宁德时代）
科创板：688xxx（如 688036 传音控股）
北交所：8xxxxx / 4xxxxx
```

### 5. 关键参与者

- **散户**：数量最多，影响市场情绪
- **公募基金**：规模最大的机构投资者
- **外资（北向资金）**：通过沪深港通进入，被称为"聪明钱"
- **量化机构**：程序化交易，近年规模快速增长
- **游资/热钱**：短线投机，擅长题材炒作

## 💡 示例 / 实操

### 使用 akshare 获取 A股数据

```python
import akshare as ak
import pandas as pd

# 获取上证指数历史数据（近1年）
df = ak.stock_zh_index_daily(symbol="sh000001")
print(df.tail())
print(df.dtypes)
```

**输出解读：**
```
          date    open    high     low   close    volume
2024-04-10  3055.98  3071.26  3040.73  3057.37  28569834
```

- `open`：开盘价
- `high`：最高价
- `low`：最低价
- `close`：收盘价
- `volume`：成交量（股）

### 获取单只股票数据

```python
# 获取贵州茅台（600519）近一年日K数据
df_moutai = ak.stock_zh_a_hist(
    symbol="600519",
    period="daily",
    start_date="20240101",
    end_date="20241231",
    adjust="qfq"  # 前复权
)
print(df_moutai.head())
```

### 读懂复权的含义

```python
# 三种复权方式对比
# 不复权（adjust=""）：展示当时真实价格，但有除权跳空
# 前复权（adjust="qfq"）：以现在价格为基准回溯，最常用于回测
# 后复权（adjust="hfq"）：以上市时价格为基准，价格持续增长
```

## 📝 小结

**今天学到了什么：**

1. **A股由三大交易所构成**：上交所（蓝筹/科创）、深交所（成长/创业）、北交所（创新中小）
2. **T+1 和涨跌停是 A股的核心规则**，与美股有本质区别，影响策略设计
3. **指数体系层次清晰**：沪深300（大盘）→ 中证500（中盘）→ 中证1000（小盘）
4. **akshare 是获取 A股数据的利器**，免费且数据全面

**关键术语：**
- OHLCV：Open / High / Low / Close / Volume
- 前复权 / 后复权：处理股息、拆股导致的价格不连续
- T+1：制度上限制了日内频繁交易（不同于 T+0 的美股）
- 北向资金：外资通过沪深港通流入 A股的资金，是重要观察指标
