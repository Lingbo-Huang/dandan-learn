## 📚 核心知识点

### 1. 市场结构

| 交易所 | 全称 | 定位 | 代表企业 |
|--------|------|------|----------|
| NYSE | 纽约证券交易所 | 传统蓝筹，全球最大按市值 | 伯克希尔、摩根大通、可口可乐 |
| NASDAQ | 纳斯达克证券交易所 | 科技成长，以电子撮合见长 | 苹果、微软、谷歌、英伟达 |
| AMEX | 美国证券交易所 | 中小盘、ETF主要上市地 | 多数 SPDR ETF |
| OTC | 场外交易市场 | 未上市证券 | 粉单市场、部分中概股 |

### 2. 核心交易规则

**交易时间（美东时间 ET，北京时间 = ET + 13小时）：**
```
盘前交易（Pre-market）：04:00 - 09:30 ET
正式交易：09:30 - 16:00 ET（北京时间 22:30 - 05:00）
盘后交易（After-hours）：16:00 - 20:00 ET
```
> 夏令时期间 ET+1，即北京时间 21:30 开盘

**与 A股的核心差异：**

| 对比项 | A股 | 美股 |
|--------|-----|------|
| 涨跌停 | ±10%/±20% | **无涨跌停** |
| 交易制度 | T+1 | **T+0**（当天可反复买卖）|
| 最小单位 | 1手=100股 | **1股**起买，支持碎股 |
| 交易时间 | 4小时 | 6.5小时（含盘前盘后更长）|
| 做空 | 受限 | **充分支持**（融券卖空）|
| 期权 | 有限（ETF期权）| **极度丰富**（个股期权）|
| 退市 | 较难 | **严格退市**，壳资源无价值 |

**T+0 的含义与影响：**
- 当天买入可当天卖出，灵活性极高
- 日内交易（Day Trading）合法，但 PDT 规则限制：资金 <25000美元时，5个交易日内日内交易不超过3次

### 3. 指数体系

| 指数 | 成分股 | 特点 |
|------|--------|------|
| 道琼斯工业平均指数（DJIA）| 30只 | 历史最悠久，价格加权，代表性有争议 |
| 标普500（S&P 500）| 500只 | 机构基准，市值加权，最权威 |
| 纳斯达克综合指数 | ~3000只 | 科技权重极高 |
| 纳斯达克100（NDX）| 100只 | 纳斯达克最大非金融股 |
| 罗素2000 | 2000只 | 小盘股代表 |
| VIX | — | 恐慌指数，衡量市场波动预期 |

**主要宽基 ETF：**
```
SPY  → 标普500
QQQ  → 纳斯达克100
IWM  → 罗素2000
DIA  → 道琼斯30
```

### 4. 美股的特色机制

**做市商制度（Market Maker）：**
- 提供买卖报价，保证市场流动性
- 纳斯达克以做市商为主，NYSE 以竞价为主

**熔断机制（Circuit Breaker）：**
```
标普500单日跌幅 7%  → 暂停交易 15 分钟（Level 1）
标普500单日跌幅 13% → 暂停交易 15 分钟（Level 2）
标普500单日跌幅 20% → 当日收市（Level 3）
```
> 历史：2020年3月，新冠恐慌引发美股史上连续4次熔断

**期权市场：**
- 几乎所有主流个股都有期权
- 到期日从当天（0DTE）到数年（LEAPS）不等
- 期权的隐含波动率（IV）是重要的市场情绪指标

### 5. 中概股（ADR）

大量中国企业通过 ADR（美国存托凭证）在美上市：
```
BABA  阿里巴巴
JD    京东
PDD   拼多多
BIDU  百度
NIO   蔚来
```

## 💡 示例 / 实操

### 使用 yfinance 获取美股数据

```python
import yfinance as yf
import pandas as pd

# 下载苹果公司近一年数据
aapl = yf.download("AAPL", start="2024-01-01", end="2024-12-31")
print(aapl.head())
print(f"年涨幅：{(aapl['Close'].iloc[-1] / aapl['Close'].iloc[0] - 1) * 100:.2f}%")
```

### 获取标普500数据

```python
sp500 = yf.download("^GSPC", period="2y")
nasdaq = yf.download("^NDX", period="2y")

# 归一化对比
sp500_norm = sp500['Close'] / sp500['Close'].iloc[0]
nasdaq_norm = nasdaq['Close'] / nasdaq['Close'].iloc[0]
```

### 获取 VIX（恐慌指数）

```python
vix = yf.download("^VIX", period="1y")
print(f"VIX 当前水平：{vix['Close'].iloc[-1]:.2f}")
# VIX > 30：市场恐慌
# VIX < 20：市场平静
# VIX < 12：市场极度乐观（反向警示？）
```

## 📝 小结

**今天学到了什么：**

1. **美股是全球流动性最高的市场**，T+0 制度和无涨跌停使得策略空间远大于A股
2. **标普500是最重要的基准**，大多数机构策略以此为 benchmark
3. **VIX 是市场情绪晴雨表**，高 VIX 意味高波动/高风险溢价
4. **yfinance 是美股数据的便捷入口**，免费、易用，适合个人研究
5. **期权市场极度丰富**，后续量化策略可以利用期权构建更复杂的收益结构

**关键术语：**
- T+0：当天买卖，日内交易合法
- PDT 规则：保证金账户 <25000$ 时，日内交易次数受限
- VIX：标普500期权隐含波动率，又称恐慌指数
- ADR：美国存托凭证，中概股在美上市的主要形式
- 熔断：市场极端下跌时的自动暂停机制
