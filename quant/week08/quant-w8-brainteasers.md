# D3 · 脑筋急转弯与逻辑题

> 这类题考察的是面对不确定性时的逻辑推理能力，而不是死记答案。

---

## 1. 经典数字游戏

### Q1：100个球，两人轮流取，每次1-5个，最后取完者赢。先手必胜策略？

```
解题思路：逆推
- 只剩 1-5 个球：先手取完，必胜
- 剩 6 个球：先手取 k 个，后手取 6-k 个，后手总能取完 → 先手必败
- 剩 7-11 个球：先手取走让对方面对 6 → 先手必胜
- 剩 12 个球：先手必败（类似 6 的逻辑）

规律：剩余球数为 6 的倍数时，当前取球者必败

100 = 6 × 16 + 4
先手取 4 个，剩 96（= 6×16），后手必败！

策略：先手取 4 个，之后每次对方取 k 个，先手取 6-k 个，保持 6 倍数。
```

### Q2：水壶问题——只有3L和5L水壶，量出4L水

```
步骤：
1. 装满5L壶
2. 5L壶倒入3L壶，5L壶剩2L
3. 倒掉3L壶的水
4. 把5L壶的2L倒入3L壶
5. 装满5L壶
6. 5L壶倒入3L壶（3L壶还能装1L），5L壶剩4L ✓

答案：4L在5L壶中。
```

## 2. 概率谜题

### Q3：飞机座位问题

100 人排队上飞机，第一个人随机坐，其余人：
- 自己位置没被占 → 坐自己的位置
- 被占 → 随机选剩余空座

问：最后一个人坐到自己位置的概率？

```
答案：1/2

直觉推导：
- 第一个人的乱坐只有两个"重要座位"：1号座和100号座
- 无论中间发生什么，当轮到第 n 个人时（n < 100），
  他不是坐自己的座位，就是继续"传递"随机选择
- 对最后一个人来说，只有两种情况：
  1. 他的座位（100号）空着：P = 1/2
  2. 1号座空着，他只能坐那里：P = 1/2

结论：P = 1/2，对称性决定的。
```

### Q4：两个孩子问题

"我有两个孩子，至少一个是男孩。" 问另一个也是男孩的概率？

```
样本空间（不考虑顺序）：{男男, 男女, 女男}（排除女女）
其中"两个都是男孩"的：{男男}

P(都是男孩 | 至少一个男孩) = 1/3

变体："我有两个孩子，大的是男孩。"
样本空间变为：{男男, 男女}
P = 1/2

直觉陷阱：信息的具体程度不同，概率不同！
```

### Q5：三门问题再延伸

N 扇门，选 1，主持人打开 k 扇空门（k < N-1），换门后的中奖概率？

```
原来选中的概率 = 1/N
换门后的中奖概率 = (N-1)/N × 1/(N-1-k) = 1/(N-1-k) × (N-1)/N

当 N=3, k=1：换后概率 = 1/1 × 2/3 = 2/3（经典蒙提霍尔）
当 N=100, k=98：换后概率 = 1/1 × 99/100 = 99/100（必换！）
```

## 3. 市场逻辑题

### Q6：如果你知道 3 个月后某股票必然涨，你今天应该怎么做？

```
标准答案：今天就买！
但面试官想听到的深度分析：

1. 交易成本：3个月持仓成本 vs. 预期收益
2. 资金成本：占用资金的机会成本
3. 信息确定性：怎么知道"必然涨"？是否合规？
4. 市场容量：买入规模会推高价格，影响收益
5. 风险管理：是否需要对冲掉其他风险（市场风险等）

量化角度：alpha = 总收益 - 市场 beta 收益 - 成本
如果总收益不够覆盖这些，不应该做。
```

### Q7：为什么期权隐含波动率往往高于实现波动率？

```
核心原因：
1. 风险溢价：卖出波动率的人要补偿尾部风险
2. 凸性：期权的 Gamma 特性让买方在极端行情下收益
3. 信息不对称：有信息的人可能买期权做保护
4. 供需：对冲需求（保险）使隐含波动率高于实际

量化意义：系统性做空波动率（Vega-neutral）在长期有正收益，
但会在危机时期暴亏（2008、2020）。
```

## 4. 思维框架题

### Q8：如何估算北京市有多少台钢琴？

**费米估算法**（面试中很常见）：

```
步骤：
1. 北京人口：约 2000 万
2. 平均家庭规模：3人/户 → 约 667 万户
3. 学钢琴比例：大约 2-3%（儿童多）→ 约 15 万户有钢琴
4. 公共场合（学校、琴行、餐厅）：约 5000 台
5. 总计：约 15-20 万台

答题技巧：
- 拆解到可估算的子问题
- 给出假设并解释
- 数量级对即可，不要追求精确
- 上下界验证合理性
```

## 小结

| 题型 | 解题策略 |
|------|---------|
| 博弈/取球 | 逆推法，寻找必败态 |
| 概率谜题 | 写出完整样本空间，再计算 |
| 条件概率 | 贝叶斯公式，注意基率 |
| 费米估算 | 分解子问题，逐步相乘 |
| 市场逻辑 | 多维度思考（成本、风险、合规）|
