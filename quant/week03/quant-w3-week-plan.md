## 本周主题

概率与统计是量化交易的数学基础。因子有效性、回测显著性、风险度量——无一不依赖统计工具。

| 天 | 主题 |
|----|------|
| D1 | 描述统计：均值、方差、偏度、峰度 |
| D2 | 概率分布：正态、t分布、厚尾分布 |
| D3 | 假设检验：t检验、p值、显著性 |
| D4 | 相关分析：Pearson / Spearman / 协方差矩阵 |
| D5 | 线性回归：最小二乘、多重共线性、因子检验 |
| D6 | 实战：用统计工具分析真实股票数据 |
| D7 | 综合：构建统计显著的因子 |
