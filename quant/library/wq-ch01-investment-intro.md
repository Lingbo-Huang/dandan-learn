## 1.1 什么是投资

### 经济意义上的投资

从经济学的广泛意义上讲，**投资是为获得一定的预期社会经济效益而进行的资金或资本物的投入及其活动过程**。

投资者将资金用于生产经营某项事业，期望获得未来收益的行为活动，包括：

1. **实物投资（实物资产投资）**：将资金用于购置和建造固定资产和流动资产。如建厂房、买设备。

2. **金融投资（金融资产投资）**：将资金用于存款、贷款，或购买股票、债券、基金等有价证券，以期获得未来价值增值收益。

## 1.2 股票投资的基本流程

股票投资的一般流程：

```
开户（券商账户）
    ↓
资金准备（银证转账）
    ↓
研究分析（基本面/技术面/量化）
    ↓
下单交易（竞价/委托）
    ↓
持仓管理（止盈止损、仓位管理）
    ↓
绩效评估（收益、风险、夏普比率）
```

**A股交易规则**：
- 交易时间：9:30–11:30，13:00–15:00（北京时间）
- 涨跌停限制：普通股±10%，科创板/创业板±20%
- T+1交收：当日买入，次日才能卖出
- 手续费：印花税0.1%（卖出），佣金约0.025%

## 1.4 什么是量化投资

**量化投资（Quantitative Investment）**：利用数学、统计学和计算机技术，通过建立数量化模型来指导投资决策的投资方式。

**核心特征**：

| 特征 | 说明 |
|------|------|
| **系统性** | 严格按照模型信号操作，排除主观情绪 |
| **纪律性** | 有明确的买卖规则，严格执行 |
| **可回测** | 历史数据可以验证策略有效性 |
| **可扩展** | 同时管理多个股票/策略，规模效应 |

**量化 vs 主观投资**：

```
主观投资：
  信息 → 研究员分析（主观判断）→ 投资决策

量化投资：
  数据 → 模型（算法）→ 信号 → 自动执行
```

## 1.5 量化投资的历史发展

| 时期 | 里程碑 |
|------|--------|
| 1950年代 | Markowitz 提出现代投资组合理论（MPT） |
| 1960年代 | CAPM 模型（Sharpe/Lintner），β 概念 |
| 1970年代 | Black-Scholes 期权定价模型 |
| 1980年代 | 统计套利兴起，D.E. Shaw、Renaissance Technologies 成立 |
| 1990年代 | 高频交易（HFT）萌芽，因子模型（Fama-French三因子）成熟 |
| 2000年代 | 量化对冲基金爆发，AQR、Two Sigma 等崛起 |
| 2010年代 | 机器学习引入量化，另类数据（卫星、NLP）兴起 |
| 2020年代 | LLM + 量化，AI Agent 策略研究 |

**文艺复兴大奖章基金**：1988-2018年，年化收益 66%（税前），堪称量化投资的最高峰。

## 1.6 量化投资的一般流程

```
1. 投资宇宙确定
   → 选什么股票池？沪深300？全A股？

2. 因子研究
   → 找到能预测收益的信号（因子）

3. 模型构建
   → 多因子模型、机器学习模型

4. 回测验证
   → 历史数据检验，评估绩效和风险

5. 实盘交易
   → 资金管理、执行优化、风险控制

6. 策略监控
   → 实时监控，定期再评估
```
